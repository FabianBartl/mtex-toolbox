Comparison of Uniformity

In order to compare the uniformity between the different grids we first perform a density estimation.

{% highlight matlab %} d(1) = grid1.calcDensity('halfwidth',5*degree); d(2) = grid2.calcDensity('halfwidth',5*degree); d(3) = grid3.calcDensity('halfwidth',5*degree); d(4) = grid4.calcDensity('halfwidth',5*degree) plot(d,'upper') mtexColorbar {% endhighlight %} {% highlight plaintext %} d = S2FunHarmonic size: 4 x 1 bandwidth: 48 isReal: true {% endhighlight %}
{% include inline_image.html file="VectorGrids_05.png" %}

We visually observe that there are quite some differences between the grids. We may also quantify the different to the uniform distribution by computing

{% highlight matlab %} norm(d-1) {% endhighlight %} {% highlight plaintext %} ans = 4.0141 0.0317 0.0426 0.0201 {% endhighlight %}

or

{% highlight matlab %} sum(abs(d-1)) {% endhighlight %} {% highlight plaintext %} ans = 5.7668 0.0600 0.0674 0.0320 {% endhighlight %}
