--- title: Fibres of Orientations last_updated: 11-Oct-2019 sidebar: documentation_sidebar permalink: OrientationFibre.html folder: documentation toc: false --- Fibres of Orientationsedit page

A fibre in orientation space is essentially a line connecting two orientations and can be represented in MTEX by a single variable of type fibre. To illustrate the definition of a fibre we first define cube and goss orientation

{% highlight matlab %} % define crystal and specimen symmetry cs = crystalSymmetry('432'); ss = specimenSymmetry('1'); % and two orientations ori1 = orientation.cube(cs,ss); ori2 = orientation.goss(cs,ss); {% endhighlight %}

and then the fibre connecting both orientations

{% highlight matlab %} f = fibre(ori1,ori2) {% endhighlight %} {% highlight plaintext %} f = fibre size: 1 x 1 crystal symmetry: 432 o1: (0°,0°,0°) o2: (0°,45°,0°) {% endhighlight %}

Finally we plot everything into the Euler space

{% highlight matlab %} % plot the fibre plot(f,'DisplayName','Fibre','linewidth',4,'linecolor','green') % and on top of it the orientations hold on plot(ori1,'DisplayName','CUBE','MarkerSize',12,'MarkerFaceColor','darkred','MarkerEdgeColor','k') plot(ori2,'DisplayName','GOSS','MarkerSize',12,'MarkerFaceColor','blue','MarkerEdgeColor','k') hold off legend {% endhighlight %} {% include inline_image.html file="OrientationFibre_01.png" %}

Alternatively, we may visualize the fibre also in axis angle space

{% highlight matlab %} % plot the fibre plot(f,'linecolor','green','linewidth',6,'axisAngle') % and on top of it the orientations hold on plot(ori1,'MarkerFaceColor','darkred','MarkerSize',15,'axisAngle') plot(ori2,'MarkerFaceColor','blue','MarkerSize',15) hold off {% endhighlight %} {% include inline_image.html file="OrientationFibre_02.png" %}

Obviously, f is not a full fibre. Since, the orientation space has no boundary a full fibre is best thought of as a circle that passes trough two fixed orientations. In order to define the full fibre us the option full

{% highlight matlab %} f = fibre(ori1,ori2,'full') hold on plot(f,'linecolor','gold','linewidth',3,'project2FundamentalRegion') hold off {% endhighlight %} {% highlight plaintext %} f = fibre size: 1 x 1 crystal symmetry: 432 o1: (0°,0°,0°) h: (100) {% endhighlight %} {% include inline_image.html file="OrientationFibre_03.png" %}

Defining a fibre by directions

Alternatively, a fibre can also be defined by a pair of a crystal and a specimen direction. In this case it consists of all orientations that alignes the crystal direction parallel to the specimen direction. As an example we define the fibre of all orientations such that the c-axis (001) is parallel to the z-axis by

{% highlight matlab %} f = fibre(Miller(0,0,1,cs),vector3d.Z) plot(f,'linecolor','gold','linewidth',4,'project2FundamentalRegion','axisAngle') {% endhighlight %} {% highlight plaintext %} f = fibre size: 1 x 1 crystal symmetry: 432 o1: (0°,0°,0°) h: (001) {% endhighlight %} {% include inline_image.html file="OrientationFibre_04.png" %}

If both directions of type Miller the fibre corresponds to all misorientations which have these two direcetion parallel.

Finally, a fibre can be defined by an initial orientation ori1 and a direction h, i.e., all orientations of this fibre satisfy

{% highlight matlab %} ori = ori1 * rot(h,omega) {% endhighlight %}

and

{% highlight matlab %} ori * h = ori1 * h {% endhighlight %}

The following code defines a fibre that passes through the cube orientation and rotates about the (111) axis.

{% highlight matlab %} f = fibre(ori1,Miller(1,1,1,cs)) plot(f,'linecolor','darkred','linewidth',4,'project2FundamentalRegion','axisAngle') {% endhighlight %} {% highlight plaintext %} f = fibre size: 1 x 1 crystal symmetry: 432 o1: (0°,0°,0°) h: (111) {% endhighlight %} {% include inline_image.html file="OrientationFibre_05.png" %}

predefined fibres

MTEX includes also a list of predefined fibres, e.g., alpha-, beta-, gamma-, epsilon-, eta- and tau fibres. Those can be defined by

{% highlight matlab %} beta = fibre.beta(cs,'full'); {% endhighlight %}

Note, that it is now straight forward to define a corresponding fibre ODF by

{% highlight matlab %} odf = fibreODF(beta,'halfwidth',10*degree) % and plot it in 3d plot3d(odf) % this adds the fibre to the plots hold on plot(beta.symmetrise,'lineColor','b','linewidth',4) hold off {% endhighlight %} {% highlight plaintext %} odf = ODF crystal symmetry : 432 specimen symmetry: 1 Fibre symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° fibre: (---) - -0.23141,-0.23141,0.94494 weight: 1 {% endhighlight %} {% include inline_image.html file="OrientationFibre_06.png" %}

Visualize an ODF along a fibre

{% highlight matlab %} plot(odf,fibre.gamma(cs)) {% endhighlight %} {% include inline_image.html file="OrientationFibre_07.png" %}

Compute volume of fibre portions

{% highlight matlab %} 100 * volume(odf,beta,10*degree) {% endhighlight %} {% highlight plaintext %} ans = 54.3540 {% endhighlight %}
