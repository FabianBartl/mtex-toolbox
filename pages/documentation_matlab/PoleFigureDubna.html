This example demonstrates the most important MTEX tools for analysing Pole Figure Data.

{% highlight matlab %} % specify crystal and specimen symmetry CS = crystalSymmetry('-3m',[1.4 1.4 1.5]); SS = specimenSymmetry; % specify file names fname = {... fullfile(mtexDataPath,'PoleFigure','dubna','Q(10-10)_amp.cnv'),... fullfile(mtexDataPath,'PoleFigure','dubna','Q(10-11)(01-11)_amp.cnv'),... fullfile(mtexDataPath,'PoleFigure','dubna','Q(11-22)_amp.cnv')}; % specify crystal directions h = {Miller(1,0,-1,0,CS),... [Miller(0,1,-1,1,CS),Miller(1,0,-1,1,CS)],... % superposed pole figures Miller(1,1,-2,2,CS)}; % specify structure coefficients c = {1,[0.52 ,1.23],1}; % import data pf = PoleFigure.load(fname,h,CS,SS,'interface','dubna','superposition',c); plot(pf) mtexColorbar {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_01.png" %}

Extract information from imported pole figure data

get raw data Data stored in a PoleFigure variable can be extracted by

{% highlight matlab %} I = pf.intensities; % intensities h = pf.h; % Miller indice r = pf.r; % specimen directions {% endhighlight %}

basic statistics There are also some basic statics on pole figure intensities

{% highlight matlab %} min(pf) max(pf) isOutlier(pf); {% endhighlight %} {% highlight plaintext %} ans = 0 0 0 ans = 1.0e+03 * 0.0898 1.3600 0.9620 {% endhighlight %}

Manipulate pole figure data

{% highlight matlab %} pf_modified = pf(pf.r.theta < 70*degree | pf.r.theta > 75*degree) plot(pf_modified) {% endhighlight %} {% highlight plaintext %} pf_modified = PoleFigure crystal symmetry : -3m1, X||a*, Y||b, Z||c* specimen symmetry: 1 h = (10-10), r = 1 x 1224 points h = (01-11)(10-11), r = 1 x 1224 points h = (11-22), r = 1 x 1224 points {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_02.png" %} {% highlight matlab %} rot = rotation.byAxisAngle(xvector-yvector,25*degree); pf_modified = rotate(pf,rot) plot(pf_modified) {% endhighlight %} {% highlight plaintext %} pf_modified = PoleFigure crystal symmetry : -3m1, X||a*, Y||b, Z||c* specimen symmetry: 1 h = (10-10), r = 72 x 19 points h = (01-11)(10-11), r = 72 x 19 points h = (11-22), r = 72 x 19 points {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_03.png" %}

PDF - to - ODF Reconstruction

{% highlight matlab %} rec = calcODF(pf,'RESOLUTION',10*degree,'iter_max',6) plotPDF(rec,h) mtexColorbar {% endhighlight %} {% highlight plaintext %} rec = ODF crystal symmetry : -3m1, X||a*, Y||b, Z||c* specimen symmetry: 1 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 2427 orientations, resolution: 10° weight: 1 {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_04.png" %} {% highlight matlab %} odf = SantaFe % define specimen directions r = regularS2Grid('antipodal') {% endhighlight %} {% highlight plaintext %} odf = ODF crystal symmetry : m-3m specimen symmetry: 222 Uniform portion: weight: 0.73 Radially symmetric portion: kernel: van Mises Fisher, halfwidth 10° center: (296.6°,48.2°,26.6°) weight: 0.27 r = S2Grid size: 72 x 19 {% endhighlight %}

define crystal directions

{% highlight matlab %} h = [Miller(1,0,0,odf.CS),Miller(1,1,0,odf.CS),Miller(1,1,1,odf.CS)]; {% endhighlight %}

simulate pole figure data

{% highlight matlab %} pf_SantaFe = calcPoleFigure(SantaFe,h,r); {% endhighlight %}

estimate an ODF with ghost correction

{% highlight matlab %} rec = calcODF(pf_SantaFe,'RESOLUTION',10*degree,'background',10) plot(rec,'sections',6) {% endhighlight %} {% highlight plaintext %} rec = ODF crystal symmetry : m-3m specimen symmetry: 222 Uniform portion: weight: 0.73238 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 128 orientations, resolution: 10° weight: 0.26762 {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_05.png" %}

without ghost correction

{% highlight matlab %} rec_ng = calcODF(pf_SantaFe,'RESOLUTION',10*degree,'background',10,'NoGhostCorrection') plot(rec_ng,'sections',6) {% endhighlight %} {% highlight plaintext %} rec_ng = ODF crystal symmetry : m-3m specimen symmetry: 222 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 150 orientations, resolution: 10° weight: 1 {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_06.png" %}

Error Analysis

{% highlight matlab %} calcError(pf_SantaFe,rec) calcError(pf_SantaFe,rec_ng) {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.0202 0.0261 0.0238 progress: 100% ans = 0.0358 0.0283 0.0250 {% endhighlight %}

Difference plot

{% highlight matlab %} plotDiff(pf_SantaFe,rec) {% endhighlight %} {% highlight plaintext %} progress: 100% {% endhighlight %} {% include inline_image.html file="PoleFigureDubna_07.png" %}
