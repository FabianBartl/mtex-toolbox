--- title: Plotting Spherical Functions last_updated: 31-Oct-2019 sidebar: documentation_sidebar permalink: S2FunPlotting.html folder: documentation toc: false --- Plotting Spherical Functions edit page

In this chapter various ways of plotting spherical functions are explained.

Here ew first define some example functions.

{% highlight matlab %} sF1 = S2Fun.smiley; f = @(v) 0.1*(v.theta+sin(8*v.x).*sin(8*v.y)); sF2 = S2FunHarmonic.quadrature(f, 'bandwidth', 150) {% endhighlight %} {% highlight plaintext %} sF2 = S2FunHarmonic bandwidth: 150 {% endhighlight %}

Smooth Plot

The default plot-command is a colored plot without contours

{% highlight matlab %} plot(sF1); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_01.png" %}
  • plot(sF1) is the same as pcolor(sF1)

Contour Plot

nonfilled contour plot plots only the contour lines

{% highlight matlab %} contour(sF1, 'LineWidth', 2); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_02.png" %}

Filled Contour Plot

filled contour plot plots the contour lines

{% highlight matlab %} contourf(sF1, 'LineWidth', 2); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_03.png" %}

3D Plot

3D plot of a sphere colored accordingly to the function values.

{% highlight matlab %} plot3d(sF2); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_04.png" %}

Surface Plot

3D plot where the radius of the sphere is transformed according to the function values

{% highlight matlab %} surf(sF2); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_05.png" %}

Section Plot

Plot the intersection of the surf plot with a plane defined by a normal vector v

{% highlight matlab %} plotSection(sF2, zvector); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_06.png" %}

Spectral Plot

plotting the Fourier coefficients

{% highlight matlab %} plotSpektra(sF1); set(gca,'FontSize', 20); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_07.png" %}

The more specific plot options are covered in the respective classes.
