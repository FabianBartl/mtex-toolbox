--- title: Plotting Spherical Functions sidebar: documentation_sidebar permalink: S2FunPlotting.html folder: documentation toc: false --- Plotting Spherical Functions edit page

In this chapter various ways of plotting spherical functions are explained. We start by defining some example functions.

{% highlight matlab %} % the smiley sF1 = S2Fun.smiley; % some oscillatory function f = @(v) 0.1*(v.theta+sin(8*v.x).*sin(8*v.y)); sF2 = S2FunHarmonic.quadrature(f, 'bandwidth', 150); {% endhighlight %}

Smooth Plot

The default pcolor command generates a colored plot without contours. Then more general command which yields the same result is simply plot

{% highlight matlab %} plotx2north plot(sF1) {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_01.png" %}

Contour Plot

Filled or non-filled contour plot can be generated by the commands contourf and contour

{% highlight matlab %} contour(sF1); nextAxis(2,1) contourf(sF1); {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_02.png" %}

Free rotate-able 3D Plot

The command plot3d yield a three dimensional plot which can be rotated freely using the mouse.

{% highlight matlab %} plot3d(sF1); how2plot = plottingConvention; how2plot.north = xvector; how2plot.outOfScreen = vector3d(0,1,2); setCamera(how2plot) {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_03.png" %}

Surface Plot

3D plot where the radius of the sphere is transformed according to the function values

{% highlight matlab %} surf(sF1) axis off setCamera(how2plot) {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_04.png" %}

Section Plot

Plot the intersection of the surf plot with a plane defined by a normal vector N

{% highlight matlab %} plotSection(sF2, zvector,'color','interp','linewidth',10) colormap spring mtexTitle('Flowerpower!') {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_05.png" %}

Spectral Plot

plotting the Fourier coefficients

{% highlight matlab %} close all plotSpektra(sF1,'FontSize',15,'linewidth',2); xlim([0,40]) {% endhighlight %}
{% include inline_image.html file="S2FunPlotting_06.png" %}

The more specific plot options are covered in the respective classes.
