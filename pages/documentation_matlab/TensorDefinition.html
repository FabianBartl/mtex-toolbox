Rank zero tensors

{% highlight matlab %} M = 5; t = tensor(M,'rank',0) {% endhighlight %} {% highlight plaintext %} t = tensor rank: 0 () 5' {% endhighlight %}

Rank one tensors

{% highlight matlab %} M = [1,2,3] t = tensor(M,'rank',1) {% endhighlight %} {% highlight plaintext %} M = 1 2 3 t = tensor rank: 1 (3) 1 2 3' {% endhighlight %} {% highlight matlab %} M = tensor(vector3d.X) {% endhighlight %} {% highlight plaintext %} M = tensor rank: 1 (3) 1 0 0' {% endhighlight %}

Rank two tensors

{% highlight matlab %} M = diag([1,2,3]) t = tensor(M,'rank',2) {% endhighlight %} {% highlight plaintext %} M = 1 0 0 0 2 0 0 0 3 t = tensor rank: 2 (3 x 3) 1 0 0 0 2 0 0 0 3' {% endhighlight %} {% highlight matlab %} t = tensor(rotation.rand) {% endhighlight %} {% highlight plaintext %} t = tensor rank: 2 (3 x 3) *10^-2 85.9 -36.91 -35.47 -35.28 -92.89 11.22 -37.09 2.87 -92.82' {% endhighlight %}

Rank three tensors

{% highlight matlab %} M = [ 0 0 0 -0.67 0 4.6 ;... 2.3 -2.3 0 0 0.67 0;... 0 0 0 0 0 0] t = tensor(M,'rank',3) {% endhighlight %} {% highlight plaintext %} M = 0 0 0 -0.6700 0 4.6000 2.3000 -2.3000 0 0 0.6700 0 0 0 0 0 0 0 t = tensor rank: 3 (3 x 3 x 3) tensor in compact matrix form: 0 0 0 -0.67 0 4.6 2.3 -2.3 0 0 0.67 0 0 0 0 0 0 0' {% endhighlight %}
