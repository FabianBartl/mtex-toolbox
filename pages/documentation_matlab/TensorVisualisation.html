Visualization

The default plot for each tensor is its directional magnitude, i.e. for each direction \(x\) we plot the quantity \(Q(x) = T_ijkl x_i x_j x_k x_l\)

{% highlight matlab %} setMTEXpref('defaultColorMap',blue2redColorMap); C = stiffnessTensor.rand plot(C,'complete','upper') {% endhighlight %} {% highlight plaintext %} C = tensor (xyz) rank: 2 (3 x 3) *10^-2 88.01 62.13 13.11 86.29 19.25 91.91 23.48 86.59 47.97 {% endhighlight %}

set back the default color map.

{% highlight matlab %} setMTEXpref('defaultColorMap',WhiteJetColorMap) {% endhighlight %}

There are more specialized visualization possibilities for specific tensors, e.g., for the elasticity tensor. See section Elasticity.
