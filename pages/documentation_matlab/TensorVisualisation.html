Visualization

The default plot for each tensor is its directional magnitude, i.e. for each direction x it is plotted Q(x) = T_ijkl x_i x_j x_k x_l

{% highlight matlab %} setMTEXpref('defaultColorMap',blue2redColorMap); C = stiffnessTensor.rand plot(C,'complete','upper') {% endhighlight %} {% highlight plaintext %} C = tensor rank: 2 (3 x 3) *10^-2 91.35 85.15 11.46 10.23 82.56 59.07 48.78 91.1 99.74 {% endhighlight %} {% include inline_image.html file="TensorVisualisation_01.png" %}

set back the default color map.

{% highlight matlab %} setMTEXpref('defaultColorMap',WhiteJetColorMap) {% endhighlight %}

There are more specialized visualization possibilities for specific tensors, e.g., for the elasticity tensor. See section Elasticity Tensor.
