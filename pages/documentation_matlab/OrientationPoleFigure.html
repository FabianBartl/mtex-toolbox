Pole figures are two dimensional representations of orientations. To illustrate this we define a random orientation with trigonal crystal symmetry

{% highlight matlab %} cs = crystalSymmetry('321'); ori = orientation.rand(cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : 321, X||a*, Y||b, Z||c specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 317.164 91.3642 85.103 0 {% endhighlight %}

Starting point is a fixed crystal direction h, e.g.,

{% highlight matlab %} % the fixed crystal directions (100) h = Miller({1,0,0},cs); {% endhighlight %}

Next the specimen directions r corresponding to all crystal directions symmetrically equivalent to h are computed

{% highlight matlab %} r = ori * h.symmetrise {% endhighlight %} {% highlight plaintext %} r = vector3d size: 1 x 6 x y z 0.0536595 -0.0871034 1.15016 0.758843 -0.719476 0.48974 -0.758843 0.719476 -0.48974 -0.0536595 0.0871034 -1.15016 0.705183 -0.632373 -0.66042 -0.705183 0.632373 0.66042 {% endhighlight %}

and ploted in a spherical projection

{% highlight matlab %} plot(r) {% endhighlight %}
{% include inline_image.html file="OrientationPoleFigure_01.png" %}

Since the trigonal symmetry group has six symmetry elements the orientation or better th crytal direction (100) points towards six direction with respect to the specimen coordinate system.

A shortcut for the above computations is the command

{% highlight matlab %} % a pole figure plot plotPDF(ori,Miller({1,0,-1,0},{0,0,0,1},{1,1,-2,1},ori.CS)) {% endhighlight %}
{% include inline_image.html file="OrientationPoleFigure_02.png" %}

We observe, that for some crystal directions only the upper hemisphere is plotted while for other upper and lower hemisphere are plotted. The reason is that if h and -h are symmetrically equivalent the upper and lower hemisphere of the pole figure are symmetric as well. In such cases MTEX plots automatically only the upper hemisphere.
