returns a quaternion q with q u_1 = v1 and q u2 = v2

View Code

Description

The method vec42quat defines a [[quaternion_index.html,quaternion]] q by 4 directions u1, u2, v1 and v2 such that q * u1 = v1 and q * u2 = v2

Syntax

q = vec42quat(u1,v1,u2,v2)

Input

u1, u2

[[vector3d_index.html,vector3d]]
v1, v2

[[vector3d_index.html,vector3d]]

Output

q

[[quaternion_index.html,quaternion]]
