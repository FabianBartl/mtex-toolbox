extend EBSD data to an grid

View Code

Description

This function transforms EBSD on non regular grids into regular grids. No interpolation is done herby. Grid points in the regular grid that do not have a correspondence in the regular grid are set to NaN.

Syntax

[ebsdGrid,newId] = gridify(ebsd)

Input

ebsd

an [[EBSD_index.html,EBSD]] data set with a non regular grid

Output

ebsd

@EBSDSquare data on a regular grid
newId

closest regular grid point for every non regular grid point
