--- title: Exporting EBSD Data last_updated: 15-Nov-2020 sidebar: documentation_sidebar permalink: EBSDExport.html folder: documentation toc: false --- Exporting EBSD Data edit page

MTEX allows you to export EBSD data as h5, ctf, ang or cpr/crc files. The syntax is

{% highlight matlab %} ebsd.export('myFile.ctf') {% endhighlight %}

The file format is detected automatically from the the extension of the specified file name.

Note that information may be lost during export into a format different then the import format as not all file formats support all properties.
