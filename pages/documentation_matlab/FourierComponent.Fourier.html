(method of [[FourierComponent_index.html,FourierComponent]])

get Fourier coefficients of and ODF

Returns the Fourier coefficients of an ODF. If no option is specified all allready precomputed Fourier coefficients are returned. If the option 'order' is specified only Fourier coefficients of this specific order are returned. If the option 'bandwidth' is specified all Fourier coefficients up to this bandwidth are returned.

View Code

Syntax

f_hat = Fourier(odf,'order',L) f_hat = Fourier(odf,'bandwidth',B)

Input

odf

[[ODF_index.html,ODF]]
L

order of Fourier coefficients to be returned
B

maximum order of Fourier coefficients to be returned
param,val Parameters and values that control Fourier
Parameter Description
'l2-normalization'

used L^2 normalization

Output

f_hat

Fourier coefficient - complex (2L+1)x(2L+1) matrix
