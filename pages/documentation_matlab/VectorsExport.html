--- title: Exporting Vectors last_updated: 11-Oct-2019 sidebar: documentation_sidebar permalink: VectorsExport.html folder: documentation toc: false --- Exporting Vectorsedit page

Large lists of vectors can be imported from a text file by the command

{% highlight matlab %} fname = fullfile(mtexDataPath,'vector3d','vectors.txt'); v = vector3d.load(fname,'ColumnNames',{'polar angle','azimuth angle'}) {% endhighlight %} {% highlight plaintext %} v = vector3d size: 1000 x 1 {% endhighlight %}

and exported by the command

{% highlight matlab %} export(v,fname,'polar') {% endhighlight %}
