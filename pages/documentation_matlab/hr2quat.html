arbitrary quaternion q with q * h = r

View Code

Description

The method hr2quat defines a [[quaternion_index.html,rotation]] q by a crystal direction h and a specimen direction r such that q * h = r

Syntax

q = hr2quat(h,r)

Input

h

[[Miller_index.html,Miller]] or [[vector3d_index.html,vector3d]]
r

[[vector3d_index.html,vector3d]]

Output

q

[[quaternion_index.html,quaternion]]
