Let us consider the uncorrelated missorientation ODF corresponding to our model ODF.

{% highlight matlab %} mtexdata titanium odf = calcDensity(ebsd.orientations) {% endhighlight %} {% highlight plaintext %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 8100 (100%) Titanium (Alpha) LightSkyBlue 622 X||a, Y||b*, Z||c* Properties: ci, grainid, iq, sem_signal Scan unit : um X x Y x Z : [0 996] x [0 998] x [0 0] Normal vector: (0,0,1) odf = SO3FunHarmonic (Titanium (Alpha) → xyz) bandwidth: 25 weight: 1 {% endhighlight %} {% highlight matlab %} % the uncorrelated mdf = calcMDF(odf) {% endhighlight %} {% highlight plaintext %} mdf = SO3FunHarmonic (Titanium (Alpha) → Titanium (Alpha)) antipodal: true bandwidth: 25 weight: 1 {% endhighlight %} {% highlight matlab %} plotSection(mdf,'axisAngle') {% endhighlight %}
