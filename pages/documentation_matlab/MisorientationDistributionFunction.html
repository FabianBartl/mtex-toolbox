Let us consider the uncorrelated missorientation ODF corresponding to our model ODF.

{% highlight matlab %} mtexdata titanium odf = calcDensity(ebsd.orientations) {% endhighlight %} {% highlight plaintext %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 8100 (100%) Titanium (Alpha) LightSkyBlue 622 X||a, Y||b*, Z||c* Properties: ci, grainid, iq, sem_signal, x, y Scan unit : um odf = ODF crystal symmetry : Titanium (Alpha) (622, X||a, Y||b*, Z||c*) Harmonic portion: degree: 25 weight: 1 {% endhighlight %} {% highlight matlab %} % the uncorrelated mdf = calcMDF(odf) {% endhighlight %} {% highlight plaintext %} mdf = MDF crystal symmetry : Titanium (Alpha) (622, X||a, Y||b*, Z||c*) crystal symmetry : Titanium (Alpha) (622, X||a, Y||b*, Z||c*) antipodal: true Harmonic portion: degree: 20 weight: 1 {% endhighlight %} {% highlight matlab %} plotSection(mdf,'axisAngle') {% endhighlight %}
