Class Description

The class kernel is needed in MTEX to define the specific form of unimodal and fibre symmetric ODFs. It has to be passed as an argument when calling the methods uniformODF and fibreODF.

SUB: Defining a kernel function

A kernel is defined by specifying its name and its free parameter. Alternatively one can also specify the halfwidth of the kernel. Below you find a list of all kernel functions supported by MTEX.

{% highlight matlab %} psi{1} = AbelPoissonKernel(0.79); psi{2} = deLaValeePoussinKernel(13); psi{3} = vonMisesFisherKernel(7.5); psi{4} = bumpKernel(35*degree); psi{5} = DirichletKernel(3); psi{6} = GaussWeierstrassKernel(0.07); psi{7} = fibreVonMisesFisherKernel(7.2); psi{8} = SquareSingularityKernel(0.72); {% endhighlight %} {% highlight plaintext %} Warning: The syntax "deLaValeePoussinKernel" is obsolete. Please use "deLaValleePoussinKernel" instead. {% endhighlight %}

SUB: Plotting the kernel

Using the plot command you can plot the kernel as a function on SO(3) as well as the corresponding PDF, or its Fourier coefficients

{% highlight matlab %} % the kernel on SO(3) close; figure('position',[100,100,700,450]) hold all for i = 1:numel(psi) plot(psi{i},'DisplayName',class(psi{i})); end hold off legend(gca,'show') {% endhighlight %} {% include inline_image.html file="kernel_index_01.png" %}

the corresponding PDF

{% highlight matlab %} close; figure('position',[100,100,700,450]) hold all for i = 1:numel(psi) plotPDF(psi{i},'RK','DisplayName',class(psi{i})); end hold off ylim([-1,20]) legend(gca,'show') {% endhighlight %} {% include inline_image.html file="kernel_index_02.png" %}

the Fourier coefficients of the kernels

{% highlight matlab %} close; figure('position',[100,100,500,450]) hold all for i = 1:numel(psi) plotFourier(psi{i},'bandwidth',32,'DisplayName',class(psi{i})); end hold off legend(gca,'show') {% endhighlight %} {% include inline_image.html file="kernel_index_03.png" %}
