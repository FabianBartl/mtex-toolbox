Inverse Pole Figure

Inverse pole figures are two dimensional representations of orientations. To illustrate this we define a random orientation with trigonal crystal symmetry

{% highlight matlab %} cs = crystalSymmetry('321') ori = orientation.rand(cs) {% endhighlight %} {% highlight matlab %} cs = crystalSymmetry symmetry : 321 a, b, c : 1, 1, 1 reference frame: X||a*, Y||b, Z||c* ori = orientation size: 1 x 1 crystal symmetry : 321, X||a*, Y||b, Z||c* specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 324.088 85.8333 24.0304 0 {% endhighlight %}

Starting point is a fixed specimen direction r, e.g.,

{% highlight matlab %} % the fixed crystal directions z r = vector3d.Z {% endhighlight %} {% highlight matlab %} r = vector3d size: 1 x 1 x y z 0 0 1 {% endhighlight %}

Next the crystal direction corresponding to the specimen direction r according to the orientation r are computed

{% highlight matlab %} h = inv(ori) * r {% endhighlight %} {% highlight matlab %} h = Miller size: 1 x 1 symmetry: 321, X||a*, Y||b, Z||c* h -0.1037 k 0.9109 i -0.8072 l 0.0727 {% endhighlight %}

and ploted in a spherical projection

{% highlight matlab %} plot(h.symmetrise,'fundamentalRegion') {% endhighlight %} {% include inline_image.html file="OrientationInversePoleFigure_01.png" %}

A shortcut for the above computations is the command

{% highlight matlab %} % a pole figure plot plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z]) {% endhighlight %} {% include inline_image.html file="OrientationInversePoleFigure_02.png" %}
