Inverse pole figures are two dimensional representations of orientations. To illustrate this we define a random orientation with trigonal crystal symmetry

{% highlight matlab %} cs = crystalSymmetry('321'); ori = orientation.rand(cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : 321, X||a*, Y||b, Z||c specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 63.0747 150.699 208.23 0 {% endhighlight %}

Starting point is a fixed specimen direction r, e.g.,

{% highlight matlab %} % the fixed crystal directions z r = vector3d.Z {% endhighlight %} {% highlight plaintext %} r = vector3d size: 1 x 1 x y z 0 0 1 {% endhighlight %}

Next the crystal direction h corresponding to the specimen direction r according to the orientation ori is computed

{% highlight matlab %} h = inv(ori) * r {% endhighlight %} {% highlight plaintext %} h = Miller size: 1 x 1 symmetry: 321, X||a*, Y||b, Z||c h 0.0151 k -0.4312 i 0.4161 l -0.8721 {% endhighlight %}

and ploted in a spherical projection

{% highlight matlab %} plot(h.symmetrise,'fundamentalRegion') {% endhighlight %}
{% include inline_image.html file="OrientationInversePoleFigure_01.png" %}

A shortcut for the above computations is the command plotIPDF

{% highlight matlab %} % an inverse pole figure plot plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z]) {% endhighlight %}
{% include inline_image.html file="OrientationInversePoleFigure_02.png" %}

Contour plots

{% highlight matlab %} plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z],'contourf') mtexColorbar {% endhighlight %}
