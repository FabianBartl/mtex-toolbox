In this section we discus how to analyze grain boundary by intersecting them with certain lines. Lets start by importing some EBSD data and computing grain boundaries.

{% highlight matlab %} % load some example data mtexdata twins % detect grains [grains,ebsd.grainId,ebsd.mis2mean] = calcGrains(ebsd('indexed')); % smooth them grains = grains.smooth; % visualize the grains plot(grains,grains.meanOrientation) % extract all grain boundaries gB = grains.boundary; hold on plot(gB,'LineWidth',2) hold off {% endhighlight %} {% highlight matlab %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 46 (0.2%) notIndexed 1 22833 (100%) Magnesium LightSkyBlue 6/mmm X||a*, Y||b, Z||c* Properties: bands, bc, bs, error, mad, x, y Scan unit : um I'm going to colorize the orientation data with the standard MTEX colorkey. To view the colorkey do: colorKey = ipfColorKey(ori_variable_name) plot(colorKey) {% endhighlight %} {% include inline_image.html file="BoundaryIntersections_01.png" %}

Let start by defining some line by its endpoints and plot in on top of the boundary plot

{% highlight matlab %} xy1 = [0,10]; % staring point xy2 = [31,41]; % end point line([xy1(1);xy2(1)],[xy1(2);xy2(2)],'linestyle',':','linewidth',4,'color','white') {% endhighlight %} {% include inline_image.html file="BoundaryIntersections_02.png" %}

The command intersect computes the intersections of all boundary segments with the given line

{% highlight matlab %} [x,y] = grains.boundary.intersect(xy1,xy2); hold on scatter(x,y,'blue','linewidth',2) hold off % find the number of intersection points sum(~isnan(x)) {% endhighlight %} {% highlight matlab %} ans = 10 {% endhighlight %} {% include inline_image.html file="BoundaryIntersections_03.png" %}
