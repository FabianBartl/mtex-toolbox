--- title: MTEX vs. Bunge Convention last_updated: 07-Nov-2019 sidebar: documentation_sidebar permalink: MTEXvsBungeConvention.html folder: documentation toc: false --- MTEX vs. Bunge Convention edit page

For historical reasons MTEX defines orientations in a slightly different way than they have been defined by Bunge. As explained in topic orientations MTEX defines them as coordinate transformations from the crystal reference frame into the specimen reference frame. In contrast to this Bunge orientations are coordinate transformations from the specimen reference frame into the crystal reference frame. Lets demonstrate this by a simple example:

{% highlight matlab %} % consider cubic symmetry cs = crystalSymmetry('cubic') % and a random orientation ori = orientation.rand(cs) {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: m-3m a, b, c : 1, 1, 1 ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 18.211 80.7939 30.1013 0' {% endhighlight %}

This is now an MTEX orientation and can be used to translate crystal coordinates, i.e., Miller indices into specimen coordinates,

{% highlight matlab %} % either by multiplying from the left r = ori * Miller({1,0,0},cs) % or using the command rotate rotate(Miller({1,0,0},cs),ori) {% endhighlight %} {% highlight plaintext %} r = vector3d size: 1 x 1 x y z 0.796731 0.346591 0.49507 ans = vector3d size: 1 x 1 x y z 0.796731 0.346591 0.49507' {% endhighlight %}

A Bunge orientation is exactly the inverse of an MTEX orientation, i.e.,

{% highlight matlab %} ori_Bunge = inv(ori) {% endhighlight %} {% highlight plaintext %} ori_Bunge = inverse orientation size: 1 x 1 specimen symmetry: 1 crystal symmetry : m-3m Bunge Euler angles in degree phi1 Phi phi2 Inv. 149.899 80.7939 161.789 0' {% endhighlight %}

and translates specimen coordinates into Miller indices

{% highlight matlab %} ori_Bunge * r {% endhighlight %} {% highlight plaintext %} ans = Miller size: 1 x 1 symmetry: m-3m h 1 k 0 l 0' {% endhighlight %}

Euler angles

Since the Euler angles are the most common way to describe orientations MTEX implements them such that the Euler angles of an MTEX orientation coincide with the Euler angles of a Bunge orientation. Thus the Euler angles of orientations in MTEX agree with the Euler angles reported by all common EBSD devices, simulation software, text books and paper.

Matrix notation

Due to the above explained inverse relationship of orientations defined in MTEX and in Bunge convention, a matrix generated from an orientation in MTEX is the inverse, or equivallently, the transpose of the matrix in Bunge notation.

{% highlight matlab %} ori.matrix ori_Bunge.matrix^(-1) ori_Bunge.matrix' {% endhighlight %} {% highlight plaintext %} ans = 0.7967 -0.5197 0.3085 0.3466 -0.0253 -0.9377 0.4951 0.8540 0.1600 ans = 0.7967 -0.5197 0.3085 0.3466 -0.0253 -0.9377 0.4951 0.8540 0.1600 ans = 0.7967 -0.5197 0.3085 0.3466 -0.0253 -0.9377 0.4951 0.8540 0.1600' {% endhighlight %}

Misorientations

Since, MTEX orientations translates crystal to specimen coordinates misorientations are defined by the formula

{% highlight matlab %} ori1 = orientation.rand(cs); ori2 = orientation.rand(cs); mori = inv(ori1) * ori2 {% endhighlight %} {% highlight plaintext %} mori = misorientation size: 1 x 1 crystal symmetry : m-3m crystal symmetry : m-3m Bunge Euler angles in degree phi1 Phi phi2 Inv. 256.666 138.236 18.342 0' {% endhighlight %}

as they are commonly defined coordinate transformations from crystal to crystal coordinates. This formula is different to the misorientation formula for Bunge orientations

{% highlight matlab %} ori1_Bunge = inv(ori1); ori2_Bunge = inv(ori2); mori = ori1_Bunge * inv(ori2_Bunge) {% endhighlight %} {% highlight plaintext %} mori = misorientation size: 1 x 1 crystal symmetry : m-3m crystal symmetry : m-3m Bunge Euler angles in degree phi1 Phi phi2 Inv. 256.666 138.236 18.342 0' {% endhighlight %}

However, both formula result in exactly the same misorientation.

Summary

This list summarizes the differences between MTEX orientations and Bunge orientations.

  • formulas involving orientations - invert orientation
  • orientation Euler angles - unchanged
  • orientation matrix - transpose matrix
  • misorientations - unchanged
  • misorientation Euler angles - take Euler angles of inverse misorientation
