--- title: Ghost Effect Analysis last_updated: 10-Oct-2019 sidebar: documentation_sidebar permalink: PoleFigure2ODFGhostCorrection.html folder: documentation toc: false --- Ghost Effect Analysisedit page

A general problem in estimating an ODF from pole figure data is the fact that the odd order Fourier coefficients of the ODF are not present anymore in the pole figure data and therefore it is difficult to estimate them. Artifacts in the estimated ODF that are due to underestimated odd order Fourier coefficients are called ghost effects. It is known that for sharp textures the ghost effect is relatively small due to the strict non-negativity condition. For weak textures, however, the ghost effect might be remarkable. For those cases, MTEX provides the option ghost_correction which tries to determine the uniform portion of the unknown ODF and to transform the unknown weak ODF into a sharp ODF by substracting this uniform portion. This is almost the approach Matthies proposed in his book (He called the uniform portion phon). In this section, we are going to demonstrate the power of ghost correction at a simple, synthetic example.

Construct Model ODF

A unimodal ODF with a high uniform portion.

{% highlight matlab %} cs = crystalSymmetry('222'); mod1 = orientation.byEuler(0,0,0,cs); odf = 0.9*uniformODF(cs) + ... 0.1*unimodalODF(mod1,'halfwidth',10*degree) {% endhighlight %} {% highlight plaintext %} odf = ODF crystal symmetry : 222 specimen symmetry: 1 Uniform portion: weight: 0.9 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: (0°,0°,0°) weight: 0.1 {% endhighlight %}

Simulate pole figures

{% highlight matlab %} % specimen directions r = equispacedS2Grid('resolution',5*degree,'antipodal'); % crystal directions h = [Miller(1,0,0,cs),Miller(0,1,0,cs),Miller(0,0,1,cs)]; % compute pole figures pf = calcPoleFigure(odf,h,r); plot(pf) {% endhighlight %} {% include inline_image.html file="PoleFigure2ODFGhostCorrection_01.png" %}

ODF Estimation

without ghost correction:

{% highlight matlab %} rec = calcODF(pf,'noGhostCorrection','silent'); {% endhighlight %}

with ghost correction:

{% highlight matlab %} rec_cor = calcODF(pf,'silent'); {% endhighlight %}

Compare RP Errors

without ghost correction:

{% highlight matlab %} calcError(pf,rec,'RP') {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.1617 0.1600 0.1685 {% endhighlight %}

with ghost correction:

{% highlight matlab %} calcError(pf,rec_cor,'RP') {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.1547 0.1519 0.1584 {% endhighlight %}

Compare Reconstruction Errors

without ghost correction:

{% highlight matlab %} calcError(rec,odf) {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.1142 {% endhighlight %}

with ghost correction:

{% highlight matlab %} calcError(rec_cor,odf) {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.0269 {% endhighlight %}

Plot the ODFs

without ghost correction:

{% highlight matlab %} plot(rec,'sections',9,'silent','sigma') {% endhighlight %} {% highlight plaintext %} progress: 100% {% endhighlight %} {% include inline_image.html file="PoleFigure2ODFGhostCorrection_02.png" %}

with ghost correction:

{% highlight matlab %} plot(rec_cor,'sections',9,'silent','sigma') {% endhighlight %} {% highlight plaintext %} progress: 100% {% endhighlight %} {% include inline_image.html file="PoleFigure2ODFGhostCorrection_03.png" %}

radial plot of the true ODF

{% highlight matlab %} close all f = fibre(Miller(0,1,0,cs),yvector); plot(odf,f,'linewidth',2); hold all {% endhighlight %} {% include inline_image.html file="PoleFigure2ODFGhostCorrection_04.png" %}

{% highlight matlab %} plot(rec,f,'linewidth',2); {% endhighlight %}

radial plot with ghost correction:

{% highlight matlab %} plot(rec_cor,f,'linestyle','--','linewidth',2); hold off legend({'true ODF','without ghost correction','with ghost correction'}) {% endhighlight %} {% include inline_image.html file="PoleFigure2ODFGhostCorrection_05.png" %}

Calculate Fourier coefficients

Next, we want to analyze the fit of the Fourier coefficients of the reconstructed ODFs. To this end, we first compute Fourier representations for each ODF

{% highlight matlab %} odf = FourierODF(odf,25) rec = FourierODF(rec,25) rec_cor = FourierODF(rec_cor,25) {% endhighlight %} {% highlight plaintext %} odf = ODF crystal symmetry : 222 specimen symmetry: 1 Harmonic portion: degree: 25 weight: 1 rec = ODF crystal symmetry : 222 specimen symmetry: 1 Harmonic portion: degree: 25 weight: 1 rec_cor = ODF crystal symmetry : 222 specimen symmetry: 1 Harmonic portion: degree: 25 weight: 1 {% endhighlight %}

Calculate Reconstruction Errors from Fourier Coefficients

without ghost correction:

{% highlight matlab %} calcError(rec,odf,'L2') {% endhighlight %} {% highlight plaintext %} ans = 0.4926 {% endhighlight %}

with ghost correction:

{% highlight matlab %} calcError(rec_cor,odf,'L2') {% endhighlight %} {% highlight plaintext %} ans = 0.3400 {% endhighlight %}

Plot Fourier Coefficients

Plotting the Fourier coefficients of the recalculated ODFs shows that the Fourier coefficients without ghost correction oscillates much more than the Fourier coefficients with ghost correction
