Sections of the orientations space

A third possibility are two dimensional sections through the Euler angle space. The most popular type of such sections are the so called phi2 sections.

{% highlight matlab %} cs = crystalSymmetry('432') ss = specimenSymmetry('222') ori = orientation.rand(100,cs,ss) % as phi2 sections plotSection(ori,'phi2') {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: 432 a, b, c : 1, 1, 1 ss = orthorhombic specimenSymmetry ori = orientation size: 100 x 1 crystal symmetry : 432 specimen symmetry: 222 {% endhighlight %}
