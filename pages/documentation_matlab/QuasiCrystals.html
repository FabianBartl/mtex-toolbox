Quasi symmetry

By quasi crystals we mean materials with a non crystallographic symmetry group, e.g. including a pentagonal or decagonal symmetry axis. Such symmetry groups do not lead to perfect translational invariant lattices and hence are not crystals.

Setting up a non standard symmetry group can be done by specifying all its symmetry elements. To illustrate this definition lets consider the symmetry group corresponding to the dodecaeder which is generated by a 5 fold and a 3 fold symmetry axis.

{% highlight matlab %} rot5 = rotation.byAxisAngle(vector3d('polar',31.7171*degree,0),72*degree) rot3 = rotation.byAxisAngle(vector3d('polar',20.9054*degree,90*degree),120*degree) cs = crystalSymmetry.byElements([rot5,rot3]) plot(cs,'symbolSize',0.5,'projection','eangle','grid','on') {% endhighlight %} {% highlight plaintext %} rot5 = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 31.7176 35.9996 31.7176 0 rot3 = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 148.282 36.0004 328.282 0 cs = crystalSymmetry symmetry : unkwown a, b, c : 1, 1, 1 alpha, beta, gamma: 90°, 90°, 90°' {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_01.png" %}

Many of the MTEX methods do work also with quasi symmetries. E.g. we can identify the fundamental sector by

{% highlight matlab %} hold on plot(cs.fundamentalSector,'color','red') hold off {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_02.png" %}

the fundamental region is orientation space, which is exactly a dodecaeder,

{% highlight matlab %} oR = cs.fundamentalRegion plot(oR) axis off {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: unknown max angle: 44.4787° face normales: 12 vertices: 20' {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_03.png" %}

or define an approbiate color key

{% highlight matlab %} ipfKey = ipfHSVKey(cs); plot(ipfKey,'complete','upper','projection','eangle','resolution',0.5*degree) {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_04.png" %}

We may also choose a different setup where the 5 fold axis is aligned parallel to z

{% highlight matlab %} rot5 = rotation.byAxisAngle(zvector,72*degree) rot3 = rotation.byAxisAngle(vector3d('polar',37.377*degree,0) ,120*degree) cs = crystalSymmetry.byElements([rot5,rot3]) plot(cs,'symbolSize',0.5,'projection','eangle','grid','on') {% endhighlight %} {% highlight plaintext %} rot5 = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 72 0 0 0 rot3 = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 54.0001 63.4344 54.0001 0 cs = crystalSymmetry symmetry : unkwown a, b, c : 1, 1, 1 alpha, beta, gamma: 90°, 90°, 90°' {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_05.png" %}

or include the inversion

{% highlight matlab %} cs = crystalSymmetry.byElements([rot5,rot3,rotation.inversion]) plot(cs,'symbolSize',0.5,'projection','eangle','grid','on') {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry : unkwown a, b, c : 1, 1, 1 alpha, beta, gamma: 90°, 90°, 90°' {% endhighlight %}
{% include inline_image.html file="QuasiCrystals_06.png" %}
{% highlight matlab %} ipfKey = ipfHSVKey(cs); plot(ipfKey,'complete','upper','projection','eangle','resolution',0.5*degree) {% endhighlight %}
