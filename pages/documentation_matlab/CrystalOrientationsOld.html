Explains how to define crystal orientations and how to switch between different convention

Contents

General Concept

Since, orientations are essentialy rotations from a crystal into a specimen fixed reference frame they consist of three components: (1) a description of the crystal reference frame by means of a variable of type crystalSymmetry

{% highlight matlab %} % consider cubic crystal symmetry m-3m cs = crystalSymmetry('cubic') {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: m-3m a, b, c : 1, 1, 1 {% endhighlight %}

(2) a description of the specimen reference frame (optional) by means of of a variable of type specimen symmetry

{% highlight matlab %} % consider orthorhombic specimen symmetry mmm ss = specimenSymmetry('orthorhombic') {% endhighlight %} {% highlight plaintext %} ss = orthorhombic specimenSymmetry {% endhighlight %}

and (3) a rotation

{% highlight matlab %} % take random rotation rot = rotation.rand {% endhighlight %} {% highlight plaintext %} rot = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 353.273 38.294 139.147 0 {% endhighlight %}

Combining all the incredients we obtain an orientation by

{% highlight matlab %} ori = orientation(rot,cs,ss) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: mmm Bunge Euler angles in degree phi1 Phi phi2 Inv. 353.273 38.294 139.147 0 {% endhighlight %}

Bunge Euler angles - ZXZ convention

Most commonly orientations are defined in terms of Euler angles. Hereby an orientation is composed of three consecutive rotations in the crystal reference frame. The first is about the z-axis, the second about the x-axis, and the third again about the z-axis. The corresponding three angles are called Euler angles and are commonly denoted by phi1, Phi, phi2.

{% highlight matlab %} ori = orientation.byEuler(30*degree,50*degree,10*degree,cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 30 50 10 0 {% endhighlight %}

Matthies Euler angles - ZYZ

In contrast to the Bunge convention here the three rotations are taken about the z-axis, the y-axis, and the z-axis.

{% highlight matlab %} ori = orientation.byEuler(30*degree,50*degree,10*degree,'ZYZ',cs,ss) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: mmm Bunge Euler angles in degree phi1 Phi phi2 Inv. 120 50 280 0 {% endhighlight %}

Note that although the orientation was specified in Matthies Euler angles the output is in Bunge Euler angles. This can be changed by setting

{% highlight matlab %} % set display style to Matthies setMTEXpref('EulerAngleConvention','ZYZ') % display orientation ori % set back the default to Bunge setMTEXpref('EulerAngleConvention','Bunge') {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: mmm ZYZ Euler angles in degree alpha beta gamma Inv. 30 50 10 0 {% endhighlight %}

MTEX supports a varity of other Euler angle conventions which use different axes of rotation as discuses in the topic Euler angle conventions.

Axis angle paramterization

Another possibility to specify an orientation is to give its rotational axis and its rotational angle.

{% highlight matlab %} ori = orientation.byAxisAngle(xvector,30*degree,cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 0 30 0 0 {% endhighlight %}

Miller indices

There is also a Miller indice convention for defining crystal orientations.

{% highlight matlab %} ori = orientation.byMiller([1 0 0],[0 1 1],cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 135 90 90 0 {% endhighlight %}

Rotation Matrix

{% highlight matlab %} ori = orientation.byMatrix(eye(3),cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 0 0 0 0 {% endhighlight %}

Predefined orientations

MTEX includes a list of standard standard orientations. For example the identical or cube orientation can be defined by the syntax

{% highlight matlab %} ori = orientation.id(cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 0 0 0 0 {% endhighlight %}

In the samy way the following orientations can be defined: cube, brass, brass2, copper, copper2, cubeND22, cubeND45, cubeRD, goss, inverse goss, PLage, PLage2, QLage, QLage2, QLage3, QLage4, SR, SR2, SR3, SR4.

Note that a list of orientations can be defined using the same syntax as for the matlab commands ones, zeros, ..

{% highlight matlab %} ori = orientation.id(100,1,cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 100 x 1 crystal symmetry : m-3m specimen symmetry: 1 {% endhighlight %}

Random orientations

You may generate random orientations with

{% highlight matlab %} ori = orientation.rand(1000,cs) {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1000 x 1 crystal symmetry : m-3m specimen symmetry: 1 {% endhighlight %}

Specimen Symmetry

If the texture forming process has been symmetric the resulting is often as well. The resulting symmetry is then aligned with the specimen reference system. In MTEX specimen symmetry axes may only be defined parallel to the specimen axes x, y, and z. The syntax is very similar to the definition of crystal symmetries.

{% highlight matlab %} % define cubic crystal symmetry cs = crystalSymmetry('432') % and orthorhombic specimen symmetry ss = specimenSymmetry('222') {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: 432 a, b, c : 1, 1, 1 ss = orthorhombic specimenSymmetry {% endhighlight %}

Orientations that respect crystal as well as specimen symmetries are defined by passign the specimen symmetry as an additional argument after the crystal symmetry.

{% highlight matlab %} ori = orientation.brass(cs,ss) % {% endhighlight %} {% highlight plaintext %} ori = orientation size: 1 x 1 crystal symmetry : 432 specimen symmetry: 222 Bunge Euler angles in degree phi1 Phi phi2 Inv. 35 45 0 0 {% endhighlight %}

Conversion into Euler angles, matrix, quaternion or Rodrigues vector

There are methods to transform quaternion in almost any other of rotations as they are:

Plotting Orientations

Orientations can be visualized in many different ways. The most popular way are pole figures

{% highlight matlab %} ori = orientation.rand(5,cs); plotPDF(ori,Miller({1,0,0},{1,1,1},{1,1,0},cs)) {% endhighlight %} {% include inline_image.html file="CrystalOrientationsOld_01.png" %}

Other options are inverse pole figures, sections through the Euler space and three dimensional orientation plots.

{% highlight matlab %} % in Euler angle space plot(ori,'filled') {% endhighlight %} {% include inline_image.html file="CrystalOrientationsOld_02.png" %}

in axis angle space

{% highlight matlab %} plot(ori,'axisAngle','filled') {% endhighlight %} {% include inline_image.html file="CrystalOrientationsOld_03.png" %}
