Simulate a set of pole figures for the SantaFe standard ODF, estimate an ODF and compare it to the inital SantaFe ODF.

Simulate pole figures

{% highlight matlab %} CS = crystalSymmetry('m-3m'); % crystal directions h = [Miller(1,0,0,CS),Miller(1,1,0,CS),Miller(1,1,1,CS),Miller(2,1,1,CS)]; % specimen directions r = equispacedS2Grid('resolution',5*degree,'antipodal'); % pole figures pf = calcPoleFigure(SantaFe,h,r); % add some noise pf = noisepf(pf,100); % plot them plot(pf,'MarkerSize',5) mtexColorMap LaboTeX {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_01.png" %}

ODF Estimation with Ghost Correction

{% highlight matlab %} rec = calcODF(pf) {% endhighlight %} {% highlight plaintext %} rec = ODF crystal symmetry : m-3m specimen symmetry: 222 Uniform portion: weight: 0.57871 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 1230 orientations, resolution: 5° weight: 0.42129 {% endhighlight %}

ODF Estimation without Ghost Correction

{% highlight matlab %} rec2 = calcODF(pf,'NoGhostCorrection') {% endhighlight %} {% highlight plaintext %} rec2 = ODF crystal symmetry : m-3m specimen symmetry: 222 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 1232 orientations, resolution: 5° weight: 1 {% endhighlight %}

Error analysis

{% highlight matlab %} % calculate RP error calcError(rec,SantaFe) % difference plot between meassured and recalculated pole figures plotDiff(pf,rec) {% endhighlight %} {% highlight plaintext %} progress: 100% ans = 0.0794 progress: 100% {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_02.png" %}

Plot estimated pole figures

{% highlight matlab %} plotPDF(rec,pf.h,'antipodal') {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_03.png" %}

Plot estimated ODF (Ghost Corrected)

{% highlight matlab %} plot(rec,'sections',18,'resolution',5*degree,... 'contourf','FontSize',10,'silent','figSize','large','minmax') mtexColorMap white2black {% endhighlight %} {% highlight plaintext %} progress: 100% {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_04.png" %}

Plot odf

{% highlight matlab %} plot(SantaFe,'sections',18,'contourf','FontSize',10,'silent',... 'figSize','large','minmax') mtexColorMap white2black {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_05.png" %}

Plot Fourier Coefficients

{% highlight matlab %} close all; % true ODF plotFourier(SantaFe,'bandwidth',32,'linewidth',2) % keep plot for adding the next plots hold all % With ghost correction: plotFourier(rec,'bandwidth',32,'linewidth',2) % Without ghost correction: plotFourier(rec2,'bandwidth',32,'linewidth',2) legend({'true ODF','with ghost correction','without ghost correction'}) % next plot command overwrites plot hold off {% endhighlight %} {% include inline_image.html file="PoleFigureSantaFe_06.png" %}
