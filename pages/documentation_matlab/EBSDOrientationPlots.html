Basics of the plot types for individual orientations data

This section gives an overview over the possibilities that MTEX offers to visualize orientation data. Let us first load a sample EBSD data set

{% highlight matlab %} mtexdata forsterite {% endhighlight %} {% highlight matlab %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 58485 (24%) notIndexed 1 152345 (62%) Forsterite LightSkyBlue mmm 2 26058 (11%) Enstatite DarkSeaGreen mmm 3 9064 (3.7%) Diopside Goldenrod 12/m1 X||a*, Y||b*, Z||c Properties: bands, bc, bs, error, mad, x, y Scan unit : um {% endhighlight %}

and select all individual orientations of the Iron phase

{% highlight matlab %} ebsd('Fo').orientations {% endhighlight %} {% highlight matlab %} ans = orientation size: 152345 x 1 crystal symmetry : Forsterite (mmm) specimen symmetry: 1 {% endhighlight %}

Scatter Pole Figure Plot

A pole figure showing scattered points of these data figure can be produced by the command plotPDF.

{% highlight matlab %} plotPDF(ebsd('Fo').orientations,Miller(1,0,0,ebsd('Fo').CS)) {% endhighlight %} {% highlight matlab %} I'm plotting 1250 random orientations out of 152345 given orientations You can specify the the number points by the option "points". The option "all" ensures that all data are plotted {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_01.png" %}

Scatter (Inverse) Pole Figure Plot

Accordingly, scatter points in inverse pole figures are produced by the command plotIPDF.

{% highlight matlab %} plotIPDF(ebsd('Fo').orientations,xvector) {% endhighlight %} {% highlight matlab %} I'm plotting 12500 random orientations out of 152345 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_02.png" %}

Scatter Plot in ODF Sections

The plotting of scatter points in sections of the orientation space is carried out by the command plotSection. In the above examples, the number of plotted orientations was chosen automatically such that the plots not to become too crowded with points. The number of randomly chosen orientations can be specified by the option points.

{% highlight matlab %} plotSection(ebsd('Fo').orientations,'points',1000,'sigma','sections',9) {% endhighlight %} {% highlight matlab %} plotting 1000 random orientations out of 152345 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_03.png" %}

Scatter Plot in Axis Angle or Rodrigues Space

Another possibility is to plot the single orientations directly into the orientation space, i.e., either in axis/angle parameterization or in Rodrigues parameterization.

{% highlight matlab %} scatter(ebsd('Fo').orientations) {% endhighlight %} {% highlight matlab %} plot 2000 random orientations out of 152345 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_04.png" %}

Here, the optional option 'center' specifies the center of the unique region in the orientation space.

Orientation plots for EBSD and grains

Since EBSD and grain data involves single orientations, the above plotting commands are also applicable for those objects.

Let us consider some grains reconstructed from the EBSD data

{% highlight matlab %} grains = calcGrains(ebsd); {% endhighlight %}

Then the scatter plot of the individual orientations of the Iron phase in the inverse pole figure is achieved by

{% highlight matlab %} plotIPDF(ebsd('Fo').orientations,xvector,'points',1000, 'MarkerSize',3); {% endhighlight %} {% highlight matlab %} I'm plotting 1000 random orientations out of 152345 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_05.png" %}

In the same way, the mean orientations of grains can be visualized

{% highlight matlab %} hold all plotIPDF(grains('Fo').meanOrientation,xvector,'points',500, 'MarkerSize',3); hold off {% endhighlight %} {% highlight matlab %} I'm plotting 500 random orientations out of 4092 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_06.png" %}

One can also use different colors on the scatter points

{% highlight matlab %} h = [Miller(1,0,0,ebsd('Fo').CS),Miller(1,1,0,ebsd('Fo').CS)]; plotPDF(ebsd('Fo').orientations,ebsd('Fo').mad,h,'antipodal','MarkerSize',4) {% endhighlight %} {% highlight matlab %} I'm plotting 1250 random orientations out of 152345 given orientations You can specify the the number points by the option "points". The option "all" ensures that all data are plotted {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_07.png" %}

or some arbitrary data vector

{% highlight matlab %} plotSection(grains('Fo').meanOrientation,log(grains('Fo').area),... 'sigma','sections',9,'MarkerSize',10); {% endhighlight %} {% highlight matlab %} plotting 2000 random orientations out of 4092 given orientations {% endhighlight %} {% include inline_image.html file="EBSDOrientationPlots_08.png" %}

See also Scatter plots for more information about scatter plot and projections for more information on spherical projections.
