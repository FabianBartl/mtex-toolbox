Euler Angle Space

By default the function plot plots orientations in the three dimensional Bunge Euler angle space

{% highlight matlab %} % consider cubic symmetry cs = crystalSymmetry('cubic'); % generate 100 random orientations ori = orientation.rand(100,cs); % plot them plot(ori) {% endhighlight %}
{% include inline_image.html file="OrientationVisualization3d_01.png" %}

Note that the orientations are automatically projected into the fundamental region. In the case of cubic symmetry this means that the Euler angles \(\Phi\) and \(\phi_2\) are restricted to 90 degrees. If the orientations should be plotted at their specified Euler angles the option 'ignoreFundamentalRegion' has to be used.

{% highlight matlab %} plot(ori,'ignoreFundamentalRegion') {% endhighlight %}
{% include inline_image.html file="OrientationVisualization3d_02.png" %}

Axis Angle Space

Alternatively, orientations can be plotted in the three dimensional axis angle space.

{% highlight matlab %} plot(ori,'AxisAngle','markerEdgeColor',[0 0 0.8],'markerSize',8,'linewidth',2) {% endhighlight %}
{% include inline_image.html file="OrientationVisualization3d_03.png" %}

The orientations are automatically projected into its fundamental region. Again, this can be switched off with the option 'ignoreFundamentalRegion'.

{% highlight matlab %} plot(ori,'axisAngle','ignoreFundamentalRegion','markerEdgeColor',[0 0 0.8],'markerSize',8,'linewidth',2) % visualize the fundamental region hold on oR = fundamentalRegion(ori.CS,ori.SS) plot(oR,'color',[1 0.5 0.5]), hold off {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: 432 max angle: 62.7994Â° face normales: 14 vertices: 24 {% endhighlight %}
