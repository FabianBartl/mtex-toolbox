In many usecases one is interested in grid of orientations that somehow uniformely cover the orientation space. As there are many different grid there is a seperate topic orientation grids. The simplest way of generating equispaced orientations with given resolution is by the command

{% highlight matlab %} % define a crystal symmetry cs = crystalSymmetry('432') % define a grid of orientations ori = equispacedSO3Grid(cs,'resolution',5*degree) {% endhighlight %} {% highlight matlab %} cs = crystalSymmetry symmetry: 432 a, b, c : 1, 1, 1 ori = SO3Grid symmetry: "432" - "1" grid : 4958 orientations, resolution: 5° {% endhighlight %}

Lets visualize them

{% highlight matlab %} plot(ori,'axisAngle') {% endhighlight %} {% highlight matlab %} plot 2000 random orientations out of 4958 given orientations {% endhighlight %} {% include inline_image.html file="OrientationGrid_01.png" %}

Check for equidistribution

{% highlight matlab %} odf = unimodalODF(ori) plotPDF(odf,Miller({1,0,0},{1,1,0},{1,1,1},cs)) mtexColorbar {% endhighlight %} {% highlight matlab %} odf = ODF crystal symmetry : 432 specimen symmetry: 1 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: 4958 orientations, resolution: 5° weight: 1 {% endhighlight %} {% include inline_image.html file="OrientationGrid_02.png" %} {% highlight matlab %} ori = regularSO3Grid(cs,'resolution',5*degree) {% endhighlight %} {% highlight matlab %} ori = orientation size: 72 x 19 x 18 crystal symmetry : 432 specimen symmetry: 1 {% endhighlight %} {% highlight matlab %} plot(ori,'axisAngle') {% endhighlight %} {% highlight matlab %} plot 2000 random orientations out of 24624 given orientations {% endhighlight %} {% include inline_image.html file="OrientationGrid_03.png" %} {% highlight matlab %} odf = unimodalODF(ori) plotPDF(odf,Miller({1,0,0},{1,1,0},{1,1,1},cs)) mtexColorbar {% endhighlight %} {% highlight matlab %} odf = ODF crystal symmetry : 432 specimen symmetry: 1 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 10° center: Rotations: 24624 x 1 weight: 1 {% endhighlight %} {% include inline_image.html file="OrientationGrid_04.png" %}
