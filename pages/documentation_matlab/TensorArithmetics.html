Tensor Products

Tensor product are the canonical way how tensors interact with each other. As an example consider a rank 4 stiffness tensor

{% highlight matlab %} C = stiffnessTensor.load(fullfile(mtexDataPath,'tensor','Olivine1997PC.GPa')) {% endhighlight %} {% highlight plaintext %} C = stiffnessTensor unit: GPa rank: 4 (3 x 3 x 3 x 3) tensor in Voigt matrix representation: 320.5 68.2 71.6 0 0 0 68.2 196.5 76.8 0 0 0 71.6 76.8 233.5 0 0 0 0 0 0 64 0 0 0 0 0 0 77 0 0 0 0 0 0 78.7 {% endhighlight %}

Then by Hooks law the stiffness tensor acts on a strain tensor, e.g.,

{% highlight matlab %} eps = strainTensor(diag([1 0 -1])) {% endhighlight %} {% highlight plaintext %} eps = strainTensor rank: 2 (3 x 3) 1 0 0 0 0 0 0 0 -1 {% endhighlight %}

according to the formula

{% include inline_image.html file="TensorArithmetics_eq04666365713434560706.png" %}

and turns it into the stress tensor {% include inline_image.html file="TensorArithmetics_eq11373214381793991308.png" %} . In MTEX such tensor products can be computed in its mpst general form by the command EinsteinSum.

{% highlight matlab %} sigma = EinsteinSum(C,[1 2 -1 -2],eps,[-1 -2]) {% endhighlight %} {% highlight plaintext %} sigma = tensor rank: 2 (3 x 3) 248.9 0 0 0 -8.65 0 0 0 -161.9 {% endhighlight %}

here the negative numbers indicate the indices which are summed up. Each pair of equal negative numbers corresponds to one sum. The positive numbers indicate the order of the dimensions of the resulting tensor. Accordingly we can compute the outer product

{% include inline_image.html file="TensorArithmetics_eq08426592857783047995.png" %}

beween two rank one tensors

{% highlight matlab %} a = tensor.rand('rank',1) b = tensor.rand('rank',1) % by the command EinsteinSum(a,1,b,2) {% endhighlight %} {% highlight plaintext %} a = tensor rank: 1 (3) *10^-2 31.96 58.03 61.65 b = tensor rank: 1 (3) *10^-2 65.82 37.65 1.31 ans = tensor rank: 2 (3 x 3) *10^-2 21.04 12.03 0.42 38.2 21.85 0.76 40.58 23.21 0.81 {% endhighlight %}

and the inner product

$$ a \cdot b = \sum_i a_i b_i

by

{% highlight matlab %} EinsteinSum(a,-1,b,-1) {% endhighlight %} {% highlight plaintext %} ans = 0.4370 {% endhighlight %}

Here we used the inv to compute the inverse of any rank 2 or rank 4 tensor. There are shortcuts in MTEX for specific tensor products. E.g. the relation between stress and strain can be more compactly written as a double dot product

{% highlight matlab %} C * eps C : eps {% endhighlight %} {% highlight plaintext %} ans = stressTensor rank: 2 (3 x 3) 248.9 0 0 0 -8.65 0 0 0 -161.9 ans = stressTensor rank: 2 (3 x 3) 248.9 0 0 0 -8.65 0 0 0 -161.9 {% endhighlight %}

Rotating a tensor

Rotation a tensor is done by the command rotate. Let's define a rotation

{% highlight matlab %} r = rotation.byEuler(45*degree,0*degree,0*degree) {% endhighlight %} {% highlight plaintext %} r = rotation size: 1 x 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 45 0 0 0 {% endhighlight %}

Then the rotated tensor is given by

{% highlight matlab %} Trot = rotate(T1,r) plot(Trot) {% endhighlight %} {% highlight plaintext %} Trot = tensor rank: 2 (3 x 3) -0.3678 0.2418 -0.2254 -0.4348 0.8965 0.6321 0.03 1.0853 0.1326 {% endhighlight %}
