Interpolation!

{% highlight matlab %} sFTri = interp(nodes, f, 'linear'); % Very wow! norm(eval(sFTri, nodes)-f) % Plotting it is meh contourf(sFTri, 'upper'); {% endhighlight %} {% highlight plaintext %} ans = 2.4016e-15 {% endhighlight %}

Approximation?

{% highlight matlab %} sF = interp(nodes, f, 'harmonicApproximation'); % Error meh norm(eval(sF, nodes)-f) % Plot wow! contourf(sF, 'upper'); {% endhighlight %} {% highlight plaintext %} ans = 0.0427 {% endhighlight %}
