How to analyze slip transmission at grain boundaries

Burgers vector and normal directions

Consider hexagonal symmetry of alpha-Titanium

{% highlight matlab %} cs = crystalSymmetry('622',[3,3,4.7],'x||a','mineral','Titanium (Alpha)') {% endhighlight %} {% highlight matlab %} cs = crystalSymmetry mineral : Titanium (Alpha) symmetry : 622 a, b, c : 3, 3, 4.7 reference frame: X||a, Y||b*, Z||c {% endhighlight %}

Then basal slip is defined by the Burgers vector (or slip direction)

{% highlight matlab %} b = Miller(2,-1,-1,0,cs,'UVTW') {% endhighlight %} {% highlight matlab %} b = Miller size: 1 x 1 mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c) U 2 V -1 T -1 W 0 {% endhighlight %}

and the slip plane normal

{% highlight matlab %} n = Miller(0,1,-1,0,cs,'HKIL') {% endhighlight %} {% highlight matlab %} n = Miller size: 1 x 1 mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c) {% endhighlight %}

Accordingly we can define a slip system in MTEX by

{% highlight matlab %} sSBasal = slipSystem(b,n) {% endhighlight %} {% highlight matlab %} sSBasal = slipSystem mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c) size: 1 x 1 U V T W | H K I L CRSS 2 -1 -1 0 0 1 -1 0 1 {% endhighlight %}

The most important slip systems for cubic, hexagonal and trigonal crystal lattices are already implemented into MTEX. Those can be accessed by

{% highlight matlab %} sSBasal = slipSystem.basal(cs) {% endhighlight %} {% highlight matlab %} sSBasal = slipSystem mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c) size: 1 x 1 U V T W | H K I L CRSS 1 1 -2 0 0 0 0 1 1 {% endhighlight %}

Obviously, this is not the only basal slip system in hexagonal lattices. There are also symmetrically equivalent ones, which can be computed by

{% highlight matlab %} sSBasalSym = sSBasal.symmetrise('antipodal') {% endhighlight %} {% highlight matlab %} sSBasalSym = slipSystem mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c) size: 3 x 1 U V T W | H K I L CRSS 1 1 -2 0 0 0 0 1 1 1 -2 1 0 0 0 0 1 1 -2 1 1 0 0 0 0 1 1 {% endhighlight %}

The length of the burgers vector, i.e., the amount of displacment is

{% highlight matlab %} sSBasalSym.b.norm {% endhighlight %} {% highlight matlab %} ans = 3.0000 3.0000 3.0000 {% endhighlight %}
