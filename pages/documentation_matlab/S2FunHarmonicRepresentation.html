The class S2FunHarmonic is the heart of S2Fun, therefore, much effort is put into its functionality. So lets cover the mathematical basics.

The general problem is to approximate functions of the form

{% include inline_image.html file="S2FunHarmonicRepresentation_eq12688510093716008662.png" %}

For that we use linear combinations of spherical harmonics, which constitude a basis of all square-integrable functions on the sphere {% include inline_image.html file="S2FunHarmonicRepresentation_eq06984502416654360419.png" %} . The spherical harmonic of degree {% include inline_image.html file="S2FunHarmonicRepresentation_eq11319871188381094158.png" %} and order {% include inline_image.html file="S2FunHarmonicRepresentation_eq11306495728221823791.png" %} is defined by

{% include inline_image.html file="S2FunHarmonicRepresentation_eq11312637631266653094.png" %}

for {% include inline_image.html file="S2FunHarmonicRepresentation_eq00504858358480559463.png" %} being the associated Legendre-Polynomial, {% include inline_image.html file="S2FunHarmonicRepresentation_eq02013426684416792931.png" %} , and {% include inline_image.html file="S2FunHarmonicRepresentation_eq12319429999823797161.png" %} .

The first ten spherical harmonics look as follows

{% highlight matlab %} surf(S2FunHarmonic(eye(10))) {% endhighlight %}
{% include inline_image.html file="S2FunHarmonicRepresentation_01.png" %}

If you think you remember these from the Windows 2000 flowerbox screensaver, you are wrong. But they play a fundamental role in the computation of atomic orbital electron configurations and may have appeared in a chemistry book you've seen.

Now we seek for so-called Fourier-coefficients {% include inline_image.html file="S2FunHarmonicRepresentation_eq06555639083441694270.png" %} such that

{% include inline_image.html file="S2FunHarmonicRepresentation_eq05988222900342699238.png" %}

approximates our function. A basic strategy to achieve this is through least squares, where we minimize the functional {% include inline_image.html file="S2FunHarmonicRepresentation_eq01352763972028051437.png" %}

for the data nodes {% include inline_image.html file="S2FunHarmonicRepresentation_eq08064257365002422487.png" %} , {% include inline_image.html file="S2FunHarmonicRepresentation_eq07710141230993437949.png" %} , {% include inline_image.html file="S2FunHarmonicRepresentation_eq09421588773943226644.png" %} the target function values and {% include inline_image.html file="S2FunHarmonicRepresentation_eq17152385268655187882.png" %} our approximation evaluated in the given data nodes.

This can be done by the lsqr function of Matlab, which efficiently seeks for roots of the derivative of the given functional (also known as normal equation). In the process we compute the matrix-vector product with the Fourier-matrix multible times, where the Fourier-matrix is given by

{% include inline_image.html file="S2FunHarmonicRepresentation_eq17442220960550767857.png" %}

This matrix-vector product can be computed efficiently with the use of the nonequispaced spherical Fourier transform NFSFT.

We end up with the Fourier-coefficients of our approximation {% include inline_image.html file="S2FunHarmonicRepresentation_eq07585311203459331132.png" %} , which describe our approximation.

There are other variants to obtain these Fourier-coefficients, but we only wanted to give a feel for what they are for.
