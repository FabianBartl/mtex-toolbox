kernel density estimation from real valued data

Syntax

{% highlight matlab %} f = calcDensity(x) f = calcDensity(x,'range',[xmin;xmax]) {% endhighlight %} {% highlight matlab %} f = calcDensity([x,y],'range',[xmin,ymin;xmax,ymax]) f = calcDensity([x,y,z],'range',[xmin,ymin,zmin;xmax,ymax,zmax]) {% endhighlight %}

Input

x,y,z random samples as n x 1 vectors

Output

f density as griddedinterpolant
