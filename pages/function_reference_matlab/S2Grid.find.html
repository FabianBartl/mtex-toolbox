(method of S2Grid)

return index of all points in a epsilon neighborhood of a vector

Syntax

ind = find(S2G,v,epsilon) % find all points in a epsilon neighborhood of v
ind = find(S2G,v)         % find closest point

Input

S2G <a href="S2Grid_index.html">S2Grid</a>
v <a href="vector3d_index.html">vector3d</a>
epsilon double

Options

antipodal include <a href="AxialDirectional.html">antipodal symmetry</a>

Output

ind int32
