(method of slipSystem)

compute the Schmid factor

Syntax

SFfun = SchmidFactor(sS) % returns spherical function
SF = SchmidFactor(sS,v)
SF = SchmidFactor(sS,sigma)
SF = SchmidFactor(sS,sigma,'relative')

Input

sS list of <a href="slipSystem_index.html">slipSystem</a>
v <a href="vector3d_index.html">vector3d</a> list of tension direction
sigma <a href="stressTensor_index.html">stressTensor</a>

Output

SFfun size(sS) x 1 list of <a href="S2FunHarmonic_index.html">S2FunHarmonic</a>
SF size(sS) x size(sigma) matrix of Schmid factors
