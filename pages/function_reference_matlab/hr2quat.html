arbitrary quaternion q with q * h = r

Description

The method hr2quat defines a rotation q by a crystal direction h and a specimen direction r such that q * h = r

Syntax

q = hr2quat(h,r)

Input

h <a href="Miller_index.html">Miller</a> or <a href="vector3d_index.html">vector3d</a>
r <a href="vector3d_index.html">vector3d</a>

Output

q <a href="quaternion_index.html">quaternion</a>
