Syntax

   M = min(A)
   M = min(A,[],"all")
   M = min(A,[],dim)
   M = min(A,[],vecdim)
   M = min(A,[],___,missingflag)

Input

A Input array
scalar | vector | matrix | multidimensional array | table |
timetable
dim Dimension to operate along
positive integer scalar
vecdim Vector of dimensions
vector of positive integers
missingflag Missing value condition
"omitmissing" (default) | "omitnan" | "omitnat" | "omitundefined" |
"includemissing" | "includenan" | "includenat" | "includeundefined"
B Additional input array
scalar | vector | matrix | multidimensional array | table |
timetable
method Comparison method
"auto" (default) | "real" | "abs"

Output

M Minimum values
scalar | vector | matrix | multidimensional array | table
I Index
scalar | vector | matrix | multidimensional array | table
C Minimum elements from A or B
scalar | vector | matrix | multidimensional array | table |
timetable
