symmetrise a tensor according to its crystal symmetry

Syntax

{% highlight matlab %} % symmetrise according to a crystal symmetry T = symmetrise(T,cs) {% endhighlight %} {% highlight matlab %} % symmetrise transversally allong direction d T = symmetrise(T,d) {% endhighlight %}

Input

T tensor
cs symmetry
d vector3d

Output

T tensor
