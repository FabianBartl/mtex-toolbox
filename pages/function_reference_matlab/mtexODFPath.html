returns the default path to ODF-files

Syntax

mtexODFPath
getMTEXpref('ODFPath')
setMTEXpref('ODFPath',path)
getMTEXpref('ODFPath') long form of mtexODFPath
setMTEXpref('ODFPath',path) change the path to ODF files
