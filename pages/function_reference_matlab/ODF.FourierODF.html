compute FourierODF from another ODF

Description

Computes the Fourier coefficients of the ODF and store them in the returned @FourierODF. In order to get the Fourier coefficients of an ODF use calcFourier.

Syntax

{% highlight matlab %} odf = FourierODF(odf,L) {% endhighlight %}

Input

odf ODF
L order up to which Fourier coefficients are calculated

Output

odf @FourierODF where Fourier coefficients are stored for further use
