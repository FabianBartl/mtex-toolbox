The class sphericalRegion describes a region on the sphere that is bounded by small circles. For a list of normal vectors \(N_i\) and coefficients \(\alpha_i\) the region is defined as all vectors \(v\) on the sphere that satify for all \(i\) the condition

\[v \cdot N_i \le \alpha_i\]

Syntax

{% highlight matlab %} sR = sphericalRegion(N) sR = sphericalRegion(N,alpha) sR = sphericalRegion(N,'antipodal') {% endhighlight %}

Input

N
alpha double

Class Properties

N the normal of the bounding circles
alpha the cosine of the bounding circle
antipodal
