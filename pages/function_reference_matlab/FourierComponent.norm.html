caclulate texture index of ODF

The norm of an ODF f is defined as:

\[ t = -\int f(g)^2 dg\]

Syntax

{% highlight matlab %} t = norm(component,varargin) {% endhighlight %}

Input

odf ODF

Output

texture index double
