load tesselation data from neper files

Description

grain2d.load is a method to load the data from the tesselation files that neper outputs (only 2 dimesional tesselations!)

Syntax

{% highlight matlab %} grains = grain2d.load('filepath/filename.tess') {% endhighlight %}

Input

fname filename

Output

grain2d grain2d
