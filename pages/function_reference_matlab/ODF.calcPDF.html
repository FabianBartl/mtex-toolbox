(method of ODF)

calcPDF computed the PDF corresponding to an ODF

Syntax

pdf = calcPDF(odf,h)
pdf = calcPDF(odf,h,'superposition',c)
value = calcPDF(odf,h,r)
ipdf = calcPDF(odf,[],r)

Input

odf <a href="ODF_index.html">ODF</a>
h <a href="Miller_index.html">Miller</a> / <a href="vector3d_index.html">vector3d</a> crystal directions
r <a href="vector3d_index.html">vector3d</a> specimen directions

Options

superposition calculate superposed pdf

Flags

antipodal include <a href="AxialDirectional.html">antipodal symmetry</a>

Output

pdf pole density function <a href="S2FunHarmonicSym_index.html">S2FunHarmonicSym</a>
ipdf inverse pole density function <a href="S2FunHarmonicSym_index.html">S2FunHarmonicSym</a>
value double
