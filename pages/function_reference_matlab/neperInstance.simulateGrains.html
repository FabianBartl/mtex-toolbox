generating 3d neper tessellations

Syntax

{% highlight matlab %} neper=neperInstance; neper.filepath='C:\Users\user\Work\Mtex\NeperExamples' %select working folder, default: @tempdir numGrains = 100; odf = unimodalODF(orientation.rand); neper.simulateGrains(odf, numGrains) {% endhighlight %} {% highlight matlab %} ori=discreteSample(odf,numGrains) neper.simulateGrains(ori,'silent') {% endhighlight %}

Input

neper neperInstance
odf SO3Fun
numGrains number of grains
ori orientation
'silent' print log file, no console output

Output

allgrains.tess tessellation file, name specified at neper.filename3d, stored under neper.filepath
allgrains.ori orientation file, Euler-Bunge format,
ori_in.txt input orientations, Rodrigues format
