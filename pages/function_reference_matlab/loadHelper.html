helps to load data-matrix with ColumnNames restricts also data according to conventions (e.g. >4pi)

Syntax

{% highlight matlab %} loader = loadHelper( d, varargin ) {% endhighlight %}

Input

d (n x m) matrix
ColumnNames (1 x m) cell of names
Columns (1 x m) cell of ColumnNames indizes

Output

loader structure with some funs

Options

KeepNaN
Radians
passive rotation
