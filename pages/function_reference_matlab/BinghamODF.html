defines a Bingham distributed ODF

Description

BinhamODF defines a Bingham distributed ODF with A and Lambda.

Syntax

{% highlight matlab %} odf = BinghamODF(kappa,A,CS,SS) odf = BinghamODF(kappa,q,CS,SS) odf = BinghamODF(kappa,h,r,CS,SS) {% endhighlight %}

Input

kappa form parameter
A
h,r fibre
CS, SS crystal, specimen symmetry

Output

odf ODF
