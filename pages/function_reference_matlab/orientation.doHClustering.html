sort orientations into clusters

Syntax

{% highlight matlab %} [c,center] = doHCluster(ori,'numCluster',n) [c,center] = doHCluster(ori,'maxAngle',omega) {% endhighlight %}

Input

ori orientation
n number of clusters
omega maximum angle

Output

c list of clusters
center center of the clusters

Example

{% highlight matlab %} % generate orientation clustered around 5 centers cs = crystalSymmetry('m-3m'); center = orientation.rand(5,cs); odf = unimodalODF(center,'halfwidth',5*degree) ori = odf.calcOrientations(3000); {% endhighlight %} {% highlight plaintext %} odf = ODF crystal symmetry : m-3m specimen symmetry: 1 Radially symmetric portion: kernel: de la Vallee Poussin, halfwidth 5° center: Rotations: 5 x 1 weight: 1 {% endhighlight %} {% highlight matlab %} % find the clusters and its centers tic; [c,centerRec] = calcCluster(ori,'method','hierarchical','numCluster',5); toc {% endhighlight %} {% highlight plaintext %} Elapsed time is 8.748760 seconds. {% endhighlight %} {% highlight matlab %} % visualize result oR = fundamentalRegion(cs) plot(oR) {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: 432 max angle: 62.7994° face normales: 14 vertices: 24 {% endhighlight %}
