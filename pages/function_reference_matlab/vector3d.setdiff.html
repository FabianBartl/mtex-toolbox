(method of vector3d)

remove vectors v2 from a set of vectors v1

Syntax

v = setdiff(v1,v2) v = setdiff(v1,v2,'antipodal')

Input

v1, v2

vector3d

param,val Parameters and values that control setdiff
Parameter Description
'antipodal'

include antipodal symmetry

Output

v

vector3d
