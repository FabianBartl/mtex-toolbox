(method of tensor)

singular value decomposition of a rank two tensor

Syntax

S = svd(T)
[U,S,V] = svd(T)

Input

T list of M rank 2 <a href="tensor_index.html">tensor</a>

Output

S 3xM sorted singular values, starting with the largest
U 3xM left singular <a href="vector3d_index.html">vector3d</a>
V 3xM right singular <a href="vector3d_index.html">vector3d</a>
