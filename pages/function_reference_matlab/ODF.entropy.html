caclulate entropy of ODF

The entropy of an ODF f is defined as:

\[ e = - \int f(g) \ln f(g) dg\]

Syntax

{% highlight matlab %} e = entropy(odf,varargin) {% endhighlight %}

Input

odf ODF

Output

entropy double

Options

resolution resolution of the discretization
