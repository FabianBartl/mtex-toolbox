(method of BinghamS2)

Description

defines a spherical Bingham distribution with shape parameters |Z|
and principal axes |a|.
Z(1)>=Z(2)>=Z(3), Z(3)<0
Z(1)=Z(2)  rotationally symmetric unimodal distribution
Z(1)<<Z(2) partial girdle distribution

Syntax

BS2 = BinghamS2(Z,a)

Input

Z shape parameters
a principle axes <a href="vector3d_index.html">vector3d</a>

Output

BS2 <a href="BinghamS2_index.html">BinghamS2</a> spherical Bingham distribution
