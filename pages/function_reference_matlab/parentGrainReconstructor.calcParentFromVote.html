reconstruct parent orientations from boundary or triple point votes

Syntax

{% highlight matlab %} % take majority vote job.calcParentFromVote('minFit',2*degree,'minVotes',2) {% endhighlight %} {% highlight matlab %} % all votes must be equal job.calcParentFromVote('strict','minVotes',2) {% endhighlight %} {% highlight matlab %} % go by probability job.calcParentFromVote('probability','threshold',1.5*degree,'tolerance',1.5*degree,'minProb',0.6) {% endhighlight %}

Input

job parentGrainReconstructor

Output

job parentGrainReconstructor

Options

probability
strict require all votes to be equal
minFit minimum required fit
maxFit maximum second best fit
minVotes minimum number of required votes
minProb minimum probability
