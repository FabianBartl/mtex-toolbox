Poisson ratio of an elasticity tensor

Description

\[\nu = \frac{-S_{ijkl} x_i x_j y_k y_l}{S_{mnop} x_m x_n x_o x_p}\]

Syntax

{% highlight matlab %} nu = PoissonRatio(S,x,y) nu = PoissonRatio(S) % isotropic case {% endhighlight %}

Input

S elastic
x tension direction
y perpendicular direction

Output

nu Poisson ratio
