calculates rotational angle between orientations

Syntax

{% highlight matlab %} omega = angle(o) omega = angle(o,'noSymmetry') % ignore symmetry equivalents omega = angle(o1,o2) {% endhighlight %}

Input

o1, o2 orientation

Output

o1 x o2 angle (double)
