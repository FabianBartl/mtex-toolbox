Syntax

{% highlight matlab %} S2Grid(theta,rho) % fills a Sphere with N--nodes regularS2Grid('resolution',5*degree) % construct regular polar and azimuthal spacing equispacedS2Grid('resolution',5*degree) % construct equispaced nodes {% endhighlight %}

Input

nodes vector3d

Options

POINTS [nrho,ntheta] number of points to be generated
RESOLUTION resolution of a equispaced grid
HEMISPHERE 'lower', 'uper', 'complete', 'sphere', 'identified'}
THETA theta angle
RHO rho angle
MINRHO starting rho angle (default 0)
MAXRHO maximum rho angle (default 2*pi)
MINTHETA starting theta angle (default 0)
MAXTHETA maximum theta angle (default pi)

Flags

REGULAR generate a regular grid
EQUISPACED generate equidistribution
ANTIPODAL include antipodal symmetry
PLOT generate plotting grid
NO_CENTER ommit point at center
RESTRICT2MINMAX restrict margins to min / max

Example

{% highlight matlab %} regularS2Grid('points',[72 19]) {% endhighlight %} {% highlight plaintext %} ans = S2Grid size: 72 x 19 {% endhighlight %} {% highlight matlab %} regularS2Grid('resolution',[5*degree 2.5*degree]) {% endhighlight %} {% highlight plaintext %} ans = S2Grid size: 72 x 73 {% endhighlight %} {% highlight matlab %} equispacedS2Grid('resolution',5*degree,'maxrho',pi) plot(ans) {% endhighlight %} {% highlight plaintext %} ans = S2Grid size: 1 x 824 resolution: 5° {% endhighlight %} {% include inline_image.html file="S2Grid.S2Grid_01.png" %}
