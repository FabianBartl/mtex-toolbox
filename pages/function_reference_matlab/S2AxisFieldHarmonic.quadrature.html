Syntax

{% highlight matlab %} sF = S2AxisField.quadrature(v, value) sF = S2AxisField.quadrature(f) sF = S2AxisField.quadrature(f, 'bandwidth', M) {% endhighlight %}

Input

v vector3d
value vector3d (antipodal)
f function handle in vector3d

Options

M degree of the spherical harmonic (default: 128)
