find lattice alignements for arbitrary misorientations

Description

Given a misorientation mori find all face normals n1, n2 and crystal directions d1, d2, such that mori * n1 = n2 and mori * d1 = d2.

Syntax

{% highlight matlab %} matchLattice(mori) {% endhighlight %} {% highlight matlab %} [n1,n2,d1,d2] = matchLattice(mori) [n1,n2,d1,d2] = matchLattice(mori,'threshold',1*degree) [n1,n2,d1,d2] = matchLattice(mori,'maxIndex',3) {% endhighlight %}

Input

mori misorientation

Output

uvw,hkl Miller
n1,n2,d1,d2 Miller

Example

{% highlight matlab %} % revert sigma3 misorientation relationship [n1,n2,d1,d2] = round2Miller(CSL(3,crystalSymmetry('432'))) {% endhighlight %} {% highlight plaintext %} n1 = Miller (432) h k l 1 1 1 n2 = Miller (432) h k l 1 1 1 d1 = Miller (432) u v w 0 1 -1 d2 = Miller (432) u v w -1 1 0 {% endhighlight %} {% highlight matlab %} % revert back Bain misorientation ship cs_alpha = crystalSymmetry('m-3m', [2.866 2.866 2.866], 'mineral', 'Ferrite'); cs_gamma = crystalSymmetry('m-3m', [3.66 3.66 3.66], 'mineral', 'Austenite'); mori = orientation.Bain(cs_alpha,cs_gamma) [n_gamma,n_alpha,d_gamma,d_alpha] = round2Miller(mori) {% endhighlight %} {% highlight plaintext %} mori = misorientation (Ferrite â†’ Austenite) planes directions (100)||(100) [010]||[011] n_gamma = Miller (Ferrite) h k l 1 0 0 n_alpha = Miller (Austenite) h k l 1 0 0 d_gamma = Miller (Ferrite) u v w 0 1 0 d_alpha = Miller (Austenite) u v w 0 1 1 {% endhighlight %}
