Taylor factor from orientation gradient

Syntax

{% highlight matlab %} [M,b,eps] = calcInvTaylor(mori,sS) {% endhighlight %}

Input

mori misorientation
sS slipSystem list in crystal coordinates

Output

M taylor factor
b coefficients for the acive slip systems
eps strain tensor list in crystal coordinates

Example

{% highlight matlab %} % define 10 percent strain eps = 0.1 * strainTensor(diag([1 -0.75 -0.25])) {% endhighlight %} {% highlight plaintext %} eps = strainTensor rank: 2 (3 x 3) *10^-2 10 0 0 0 -7.5 0 0 0 -2.5 {% endhighlight %} {% highlight matlab %} % define a crystal orientation cs = crystalSymmetry('cubic') ori = orientation.byEuler(0,30*degree,15*degree,cs) {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: m-3m elements: 48 a, b, c : 1, 1, 1 ori = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 1 Bunge Euler angles in degree phi1 Phi phi2 Inv. 0 30 15 0 {% endhighlight %} {% highlight matlab %} % define a slip system sS = slipSystem.fcc(cs) {% endhighlight %} {% highlight plaintext %} sS = slipSystem symmetry: m-3m size: 1 x 1 u v w | h k l CRSS 0 1 -1 1 1 1 1 {% endhighlight %}
