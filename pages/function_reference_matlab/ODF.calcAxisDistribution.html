(method of ODF)

compute the axis distribution of an ODF or MDF

Syntax

value = calcAxisDistribution(odf)
adf = calcAxisDistribution(odf, a)

Input

odf <a href="ODF_index.html">ODF</a> orientation or misorientation distribution function
a <a href="vector3d_index.html">vector3d</a> rotational axis

Output

afd <a href="S2Fun_index.html">S2Fun</a> axis distribution function
value value of axis distribution function for rotational axis a
