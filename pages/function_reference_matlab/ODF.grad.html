gradient of odf at orientation ori

Syntax

{% highlight matlab %} g = odf.grad(ori) % compute the gradient {% endhighlight %} {% highlight matlab %} % go 5 degree in direction of the gradient ori_new = exp(ori,5*degree*normalize(g)) {% endhighlight %}

Input

odf ODF
ori orientation

Output

g vector3d gradient of the ODF at the orientations ori
