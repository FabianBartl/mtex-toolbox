(method of S2FunHarmonic)

global, local and pointwise minima of spherical functions

Syntax

{% highlight matlab %} [v,pos] = min(sF) % the position where the minimum is atained {% endhighlight %} {% highlight matlab %} [v,pos] = min(sF,'numLocal',5) % the 5 largest local minima {% endhighlight %} {% highlight matlab %} sF = min(sF, c) % minimum of a spherical functions and a constant sF = min(sF1, sF2) % minimum of two spherical functions sF = min(sF1, sF2, 'bandwidth', bw) % specify the new bandwidth {% endhighlight %} {% highlight matlab %} % compute the minimum of a multivariate function along dim sF = min(sFmulti,[],dim) {% endhighlight %}

Input

sF, sF1, sF2 S2Fun
sFmulti a multivariate S2Fun
c double

Options

kmax number of iterations
numLocal number of peaks to return
startingNodes vector3d
tolerance minimum distance between two peaks
resolution minimum step size
maxStepSize maximm step size

Output

v double
pos vector3d
