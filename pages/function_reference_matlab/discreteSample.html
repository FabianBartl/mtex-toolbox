take a diskrete sample from a list of vectors, orientations, grains, EBSD

Syntax

{% highlight matlab %} [obj,ind] = discreteSample(obj,points) discreteSample(v,points,'withReplacement') discreteSample(ori,points,'withReplacement') discreteSample(ebsd,points,'withoutReplacement') discreteSample(gB,points,'withoutReplacement') {% endhighlight %}

Input

gB grainBoundary
ebsd EBSD
ori orientation
v vector3d
points number of random samples

Output

obj same as first input
ind indeces of the selected subsamples

Options

withReplacement take the random sample with replacement (default)
withoutReplacement take the random sample without replacement
