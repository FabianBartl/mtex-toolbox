(method of EBSDhex)

smooth spatial EBSD data

Synatx ebsd = ebsd.gridify ebsd = smooth(ebsd)

Syntax

[ebsd,filter] = smooth(ebsd,varargin)

Input

ebsd <a href="EBSD_index.html">EBSD</a>

Example

mtexdata forsterite
ebsd = ebsd('indexed');
% segment grains
[grains,ebsd.grainId] = calcGrains(ebsd)
