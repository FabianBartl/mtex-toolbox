(method of symmetry)

get the maximum angle of the fundamental region

Syntax

{% highlight matlab %} omega = maxAngle(cs1) omega = maxAngle(cs1,cs2) omega = maxAngle(cs1,axis) {% endhighlight %}

Input

cs1, cs2 symmetry
axis axis(vector3d) of rotation
