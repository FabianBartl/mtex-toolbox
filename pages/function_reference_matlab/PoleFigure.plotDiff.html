(method of PoleFigure)

difference plot between two pole figures or an odf and a pole figure

Syntax

plotDiff(pf,pf2)
plotDiff(pf,odf)

Input

pf

PoleFigure

pf2

PoleFigure

odf

ODF

param,val Parameters and values that control plotDiff
Parameter Description
'RP'

calculate RP error
'l1'

calculate l^1 error
'l2'

calculate mean square error
