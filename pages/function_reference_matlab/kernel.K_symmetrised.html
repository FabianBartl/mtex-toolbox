(method of kernel)

evaluate kernel modulo symmetries

Description

K(q1,q2) = Sum(S) Sum(l) A_l Tr T_l(s1^-1 q1 s2)

Syntax

w = K_symmetrised(psi,q1,q2,CS,SS,varargin)

Input

psi <a href="kernel_index.html">kernel</a>
q1, q2 <a href="quaternion_index.html">quaternion</a>(s)
CS, SS crystal , specimen <a href="symmetry_index.html">symmetry</a>

Options

exact
epsilon
