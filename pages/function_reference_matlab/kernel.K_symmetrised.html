evaluate kernel modulo symmetries

Description

K(q1,q2) = Sum(S) Sum(l) A_l Tr T_l(s1^-1 q1 s2)

Syntax

{% highlight matlab %} w = K_symmetrised(psi,q1,q2,CS,SS,varargin) {% endhighlight %}

Input

psi kernel
q1, q2 quaternion(s)
CS, SS crystal , specimen symmetry

Options

exact
epsilon
