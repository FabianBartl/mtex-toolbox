returns the Ids of neighboring grains

Syntax

{% highlight matlab %} % neighbouring relationships within grains pairs = neighbors(grains) pairs = neighbors(grains1,grains2) {% endhighlight %} {% highlight matlab %} % all neighbouring relationships pairs = neighbors(grains,'full') {% endhighlight %}

Input

grains grain2d

Output

pairs index list of size N x 2
