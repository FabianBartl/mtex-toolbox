(method of grain2d)

returns the number of neighboring grains

Description

{% include inline_image.html file="grain2d.neighbors_eq09370701143650386670.png" %} is the total number of neighborhood relations (without self--reference).

pairs(i,:) give the indexes of two neighbored grains, i.e

neighbor_gr = grains(pairs(1,:))

selects two neighbored grains.

Syntax

[counts,pairs] = neighbors(grains)

Input

grains <a href="grain2d_index.html">grain2d</a>

Output

counts number of neighbors per grain
pairs index list of size N x 2
