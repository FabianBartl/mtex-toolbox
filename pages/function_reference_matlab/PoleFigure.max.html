maximum of two pole figures or the maximum of a single polefigure

Syntax

m  = max(pf)
pf = max(pf1,pf2)
pf = max(pf1,x) = max(x,pf1)

Input

pf [[PoleFigure_index.html,PoleFigure]]
pf1, pf2 [[PoleFigure_index.html,PoleFigure]]
x double

Output

m double
