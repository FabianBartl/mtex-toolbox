(method of orientation)

the misorientation vector between two orientations

Description

Mathematically, misorientation vector is the the inverse of the exponential map, hence the name log.

Syntax

m = log(mori) % the misorientation vector in crystal coordinats

% the misorientation vector in crystal coordinats
m = log(ori,ori_ref)

% the misorientation vector in specimen coordinats
v = log(ori,ori_ref,'left')
v = ori_ref .* m

Input

mori misorientation
ori <a href="orientation_index.html">orientation</a>
ori_ref <a href="orientation_index.html">orientation</a>

Output

m <a href="Miller_index.html">Miller</a>
v <a href="vector3d_index.html">vector3d</a>
