the misorientation vector between two orientations

Description

Mathematically, misorientation vector is the the inverse of the exponential map, hence the name log.

Syntax

{% highlight matlab %} m = log(mori) % the misorientation vector in crystal coordinats {% endhighlight %} {% highlight matlab %} % the misorientation vector in crystal coordinats m = log(ori,ori_ref) {% endhighlight %} {% highlight matlab %} % the misorientation vector in specimen coordinats v = log(ori,ori_ref,'left') v = ori_ref .* m {% endhighlight %}

Input

mori misorientation
ori orientation
ori_ref orientation

Output

m Miller
v vector3d
