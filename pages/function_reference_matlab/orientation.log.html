misorientation vector between two orientations

The misorientation vector can also be seen as the projection of an orientarion onto the tangential space of the orientation space centered at the orientation ori_ref. The inverse mapping from the tantential space onto the orientation space is the exponential map exp.

Syntax

{% highlight matlab %} m = log(mori) % the misorientation vector in crystal coordinats {% endhighlight %} {% highlight matlab %} % the misorientation vector in crystal coordinats m = log(ori,ori_ref) {% endhighlight %} {% highlight matlab %} % the misorientation vector in specimen coordinats v = log(ori,ori_ref,'left') v = ori_ref .* m {% endhighlight %}

Input

mori misorientation
ori orientation
ori_ref orientation

Output

m Miller
v vector3d
