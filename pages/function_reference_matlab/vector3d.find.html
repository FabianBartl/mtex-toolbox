return index of all points in a epsilon neighborhood of a vector

Syntax

{% highlight matlab %} ind = find(v,w,epsilon) % find all points out of v in a epsilon neighborhood of w ind = find(v,w) % find closest point out of v to w {% endhighlight %}

Input

v, w vector3d
epsilon double

Output

ind int32

Options

antipodal include antipodal symmetry
