(method of S2Fun)

defines a unimodal spherical function

Syntax

sF = S2Fun.unimodal
sF = S2Fun.unimodal('halfwidth',10*degree)

v = vector3d(1,1,1)
psi = S2DeLaValleePoussin('halfwidth',20*degree)
sF = S2Fun.unimodal(v,psi)
