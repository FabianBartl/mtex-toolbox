pointwise inner product

Syntax

{% highlight matlab %} d = dot(v1,v2) d = dot(v1,v2,'antipodal') {% endhighlight %}

Input

v1, v2 vector3d

Output

d double

Options

antipodal consider v1, v2 as axes
