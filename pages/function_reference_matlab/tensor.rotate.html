rotate a tensor by a list of rotations

Description

\[T_{rst} = T_{ijk} R_{ri} R_{sj} R_{tk}\]

Syntax

{% highlight matlab %} T = rotate(T,R,varargin) {% endhighlight %}

Input

T
R or rotation matrix or a list of them

Output

T rotated
