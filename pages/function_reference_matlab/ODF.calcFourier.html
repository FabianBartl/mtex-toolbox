compute Fourier coefficients of odf

Compute the Fourier coefficients of the ODF and store them in the returned ODF. In order to get the Fourier coefficients of an ODF use Fourier.

Syntax

{% highlight matlab %} odf = calcFourier(odf,L) {% endhighlight %}

Input

odf ODF
L order up to which Fourier coefficients are calculated

Output

nodf ODF where Fourier coefficients are stored for further use
