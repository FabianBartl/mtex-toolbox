--- title: ipfSpotKey last_updated: 06-Oct-2019 sidebar: function_reference_sidebar permalink: ipfSpotKey.html folder: function_reference toc: false --- ipfSpotKeyedit page

Maps an individual color to each given crystal directions being parallel to a specimen direction (fibre) Properties: center - list of crystal directions Miller color - n-by-3 list representing RGB values, one for each center psi - kernel providing the width and brightness for colored fibre inversePoleFigureDirection - specimen direction vector3d

Syntax

oM = ipfSpotKey(varargin)
