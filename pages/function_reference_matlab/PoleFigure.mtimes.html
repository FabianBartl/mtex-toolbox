scaling of PoleFigures, implements pf1 * b and a * pf2

overload the * operator, i.e. one can now write x * pf in order to scale the PoleFigure pf by the factor x

Syntax

{% highlight matlab %} pf = mtimes(arg1,arg2) {% endhighlight %}
