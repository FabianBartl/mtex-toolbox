(method of orientation)

exponential function

Syntax

ori_2 = exp(ori_1,v)
ori_2 = exp(ori_1,S)

Input

ori_1 <a href="orientation_index.html">orientation</a>
v <a href="vector3d_index.html">vector3d</a> axis of rotation scaled by the angle of rotation
S skew symmetry <a href="tensor_index.html">tensor</a>

Output

ori_2 <a href="orientation_index.html">orientation</a> rotate ori_1 about axis v with angle norm(v)
