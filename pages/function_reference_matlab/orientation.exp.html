(method of orientation)

exponential function

Syntax

ori_2 = exp(ori_1,v) ori_2 = exp(ori_1,S)

Input

ori_1

orientation

v

vector3d axis of rotation scaled by the angle of rotation
S

skew symmetry tensor

Output

ori_2

orientation rotate ori_1 about axis v with angle norm(v)
