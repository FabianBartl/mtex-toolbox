defines uniaxial tension/compression tensors

Syntax

% uniaxial tension
L = velocityGradientTensor.uniaxial(d,r)

% uniaxial compression
L = velocityGradientTensor.uniaxial(d,-r)

Input

d tension direction [[vector3d_index.html,vector3d]]
r strain rate

Output

L [[velocityGradientTensor_index.html,velocityGradientTensor]]
