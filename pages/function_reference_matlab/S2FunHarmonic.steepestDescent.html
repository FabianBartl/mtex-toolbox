calculates the minimum of a spherical harminc

Syntax

{% highlight matlab %} [v,pos] = steepestDescent(sF) % the position where the minimum is atained {% endhighlight %} {% highlight matlab %} [v,pos] = steepestDescent(sF,'numLocal',5) % the 5 largest local minima {% endhighlight %} {% highlight matlab %} % with all options [v,pos] = steepestDescent(sF, 'startingnodes') {% endhighlight %}

Output

v double
pos vector3d

Options

kmax number of iterations
numLocal number of peaks to return
startingNodes vector3d
tolerance minimum distance between two peaks
resolution minimum step size
maxStepSize maximm step size
