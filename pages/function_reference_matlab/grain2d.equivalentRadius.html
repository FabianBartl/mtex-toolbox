--- title: equivalentRadius last_updated: 01-Apr-2020 sidebar: function_reference_sidebar permalink: grain2d.equivalentRadius.html folder: function_reference toc: false --- equivalentRadius edit page

returns the equivalent radius of grain-polygon

defined as

\[ r = \sqrt{\frac{A}{\pi}} \],

where \(A\) is the area of a grain

Syntax

{% highlight matlab %} r = equivalentRadius(grains) {% endhighlight %}

Input

grains grain2d

Output

r radius (in measurement units)

See also

grain2d.deltaarea grain2d.equivalentPerimeter grain2d.paris
