sort orientations into clusters

Syntax

{% highlight matlab %} [c,center] = calcCluster(ori,'halfwidth',2.5*degree) [c,center] = calcCluster(ori,'numCluster',n,'method','hierarchical') [c,center] = calcCluster(ori,'maxAngle',omega,'method','hierarchical') {% endhighlight %}

Input

ori orientation
n number of clusters
omega maximum angle

Output

c list of clusters
center center of the clusters

Example

{% highlight matlab %} % generate orientation clustered around 5 centers cs = crystalSymmetry('432'); center = orientation.rand(5,cs); odf = unimodalODF(center,'halfwidth',5*degree) ori = odf.discreteSample(1500); {% endhighlight %} {% highlight plaintext %} odf = SO3FunRBF (432 → xyz) multimodal components kernel: de la Vallee Poussin, halfwidth 5° center: 5 orientations Bunge Euler angles in degree phi1 Phi phi2 weight 319.234 42.8325 53.1541 0.2 156.998 113.934 169.984 0.2 357.431 85.8389 146.392 0.2 24.8782 48.7613 12.565 0.2 345.464 128.886 9.31636 0.2 {% endhighlight %} {% highlight matlab %} % find the clusters and its centers [cId,centerRec] = calcCluster(ori,'silent'); {% endhighlight %} {% highlight plaintext %} Step size:1.25° Step size:0.20972° Step size:0.068719° Step size:0.054976° Step size:1.5625° Step size:0.4096° Step size:0.085899° Step size:0.054976° Step size:1.9531° Step size:0.32768° Step size:0.085899° Step size:0.054976° Step size:0.512° Step size:0.085899° Step size:3.0518° Step size:1° Step size:0.20972° Step size:0.054976° {% endhighlight %} {% highlight matlab %} % visualize result for i = 1:length(centerRec) plot(ori(cId==i),'axisAngle') hold on plot(centerRec(i),'MarkerFaceColor','k','MarkerSize',15) end hold off {% endhighlight %}
