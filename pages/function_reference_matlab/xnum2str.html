convert number to string

Syntax

s = xnum2str(n)
s = xnum2str(n,'cell')
s = xnum2str(n,'precision',2)
s = xnum2str(n,'fixedWidth',5)
s = xnum2str(n,'delimiter',', ')

Input

n double | int

Output

s string
