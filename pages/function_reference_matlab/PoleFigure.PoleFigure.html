(method of PoleFigure)

constructor

PoleFigure is the low level constructor. For importing real world data you might want to use the predefined interfaces

Syntax

{% highlight matlab %} pf = PoleFigure(h,r,intensities,varargin) {% endhighlight %}

Input

h crystal directions (vector3d | Miller)
r specimen directions (S2Grid)
intensities diffraction counts (double)
CS,SS crystal, specimen symmetry

Options

superposition weights for superposed crystal directions
background background intensities
