(method of PoleFigure)

constructor

PoleFigure is the low level constructor. For importing real world data you might want to use the predefined interfaces

Syntax

pf = PoleFigure(h,r,intensities,varargin)

Input

h crystal directions (<a href="vector3d_index.html">vector3d</a> | <a href="Miller_index.html">Miller</a>)
r specimen directions (<a href="S2Grid_index.html">S2Grid</a>)
intensities diffraction counts (double)
CS,SS crystal, specimen <a href="symmetry_index.html">symmetry</a>

Options

superposition weights for superposed crystal directions
background background intensities
