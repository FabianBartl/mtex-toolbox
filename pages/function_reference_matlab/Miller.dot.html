inner product between two Miller indece

Syntax

{% highlight matlab %} a = dot(m1,m2) a = dot(m1,m2,'antipodal') {% endhighlight %}

Input

m1,m2 Miller

Output

d double [length(m1) length(cs)]

Options

antipodal consider m1,m2 with antipodal symmetry
all
