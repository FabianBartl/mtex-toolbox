grain3d.slice is a method to intersect 3D grain data by a plane to get grain2d slices

Syntax

{% highlight matlab %} N = vector3d(1,1,1) % plane normal P0 = vector3d(0.5, 0.5, 0.5) % point within plane grain2d = grains.slice(N,P0) {% endhighlight %} {% highlight matlab %} V = vector3d([0 1 0],[0 1 1],[0 1 0]) grain2d = grains.slice(V) % set of points {% endhighlight %} {% highlight matlab %} plane = createPlane(P0,N) % note different sequence of inputs! grain2d = grains.slice(plane) % plane in matGeom format {% endhighlight %}

Input

grains grain3d
plane plane in matGeom format

Output

grain2d grain2d
