(method of grainBoundary)

implements gB(1:10)

Syntax

gB(1:10)                    % the 10 first boundaries
gB('Forsterite','Epidote')  % only Forsterite - Epidote boundaries
gB(cond)

Input

gB <a href="grainBoundary_index.html">grainBoundary</a>
cond logical array with same size as gB
