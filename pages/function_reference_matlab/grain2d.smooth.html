constraint laplacian smoothing of grain boundaries and inner boundaries.

Syntax% grains = smooth(grains,iter,varargin)

Input

grains [[grain2d_index.html,grain2d]]
iter number of iterations (optional, default: 1)

Options

'gauss','exp','umbrella' or 'rate' interpoaltion methods (default: 'rate')
second_order, S2 second order smoothing

Output

grains [[grain2d_index.html,grain2d]]
