regular grid in Euler orientation space

Syntax

{% highlight matlab %} S3G = regularSO3Grid(cs) S3G = regularSO3Grid(cs,ss,'resolution',2.5*degree) % specify the resolution S3G = regularSO3Grid(cs,ss,'resolution',5*degree,'ZYZ') % use ZYZ convention S3G = regularSO3Grid(cs,ss,'phi2','sections',10) % 10 phi2 sections S3G = regularSO3Grid {% endhighlight %}

Input

cs crystalSymmetry
ss specimenSymmetry

Output

Options

resolution Euler angle resolution
sections number of sections

Flags

phi1 | Phi | phi2 sections along which variable
