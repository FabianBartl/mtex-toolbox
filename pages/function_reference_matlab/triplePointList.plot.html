plot triple points

Syntax

{% highlight matlab %} plot(grains.triplePoints) plot(grains.triplePoints,'color','r') plot(grains('Forsterite').triplePoints,gB('Forsterite','Forsterite').misorientation.angle) {% endhighlight %}

Input

tP triplePointList

Options

marker square, diamond, circle, asterics,
markerSize size of the marker
markerEdgeColor edge color
markerFaceColor face color
markerColor both colors
linewidth edge width of the marker
