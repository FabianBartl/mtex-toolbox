(method of orientation)

define orientations by Miller Bravais indeces

Description

Defines an orientation_index.html,orientation ori by Miller indece (hkl) and [uvw] such that ori * (hkl) = 001 and ori * [uvw] = 100

Syntax

hkl = Miller(h,k,l,CS,'hkl')
uvw =  Miller(u,v,w,CS,'uvw')
ori = orientation.byMiller(hkl,uvw)

ori = orientation.byMiller([h k l],[u v w],CS)

Input

hkl, uvw

Miller

h,k,l

Miller indece (double)
u,v,w

Miller indece (double)
CS

crystalSymmetry

Output

ori

orientation
