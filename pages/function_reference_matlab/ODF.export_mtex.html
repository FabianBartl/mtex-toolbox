export an ODF into the MTEX format

View Code

Syntax

export(odf,'filename')

Input

odf

ODF to be exported
filename

name of the ascii file
param,val Parameters and values that control export_mtex
Parameter Description
'ZYZ, ABG'

Matthies (alpha, beta, gamma) convention (default)
'ZXZ,BUNGE'

Bunge (phi1,Phi,phi2) convention
