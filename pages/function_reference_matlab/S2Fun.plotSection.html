plot a section through a spherical function

Syntax

{% highlight matlab %} N = vector3d.Z; % normal direction plotSection(sF,N) % plot in equator plane {% endhighlight %} {% highlight matlab %} theta = pi/3; % polar angle plotSection(sF,N,theta) % plot small circle at 30 degree from the north pole {% endhighlight %} {% highlight matlab %} rho = linspace(0,pi); % azimuthal angle plotSection(sF,N,theta,rho) % plot half small circle at 30 degree {% endhighlight %}

Input

sF S2Fun
N normal direction of the intersection plane
theta polar angle of the intersection plane
rho azimuthal angle of the points to be plotted

Output
