draw rotations connected by lines

Syntax

{% highlight matlab %} line(rot,'linecolor','r','linewith',2) {% endhighlight %}

Input

rot list of rotation

Example

{% highlight matlab %} cs1 = crystalSymmetry('321') cs2 = crystalSymmetry('432') oR = fundamentalRegion(cs1,cs2) plot(oR) {% endhighlight %} {% highlight plaintext %} cs1 = crystalSymmetry symmetry : 321 elements : 6 a, b, c : 1, 1, 1 reference frame: X||a*, Y||b, Z||c* cs2 = crystalSymmetry symmetry: 432 elements: 24 a, b, c : 1, 1, 1 oR = orientationRegion crystal symmetry: 321, X||a*, Y||b, Z||c* crystal symmetry: 432 max angle: 56.6003Â° face normales: 14 vertices: 20 {% endhighlight %}
