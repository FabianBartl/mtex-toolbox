(method of tensor)

return tensor as a matrix

Syntax

m = matrix(T)
m = matrix(T,'Voigt')
m = matrix(T,'Kelvin')

Input

T <a href="tensor_index.html">tensor</a>

Options

Voigt give a 4 rank tensor in Voigt notation, i.e. as a 6 x 6 matrix
Kelvin same as above but with Kelvin normalization

Output

m matrix
