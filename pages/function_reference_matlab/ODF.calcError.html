(method of ODF)

calculate approximation error between two ODFs

Syntax

e = calcError(odf1,odf2)
e = calcError(odf,pf,'RP')

Input

odf1, odf2 <a href="ODF_index.html">ODF</a>
pf <a href="PoleFigure_index.html">PoleFigure</a>
S3G <a href="SO3Grid_index.html">SO3Grid</a> of quadrature nodes (optional)

Options

L0 measure of the orientation space where $|odf1 -- odf2|>\epsilon|
L1 L^1 error
L2 L^2 error
RP RP error (default)
resolution resolution used for calculation of the error
