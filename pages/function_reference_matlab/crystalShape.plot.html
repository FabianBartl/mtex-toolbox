colorize grains

Syntax

{% highlight matlab %} plot(cS) % colorize by phase plot(x,y,cS) plot(x,y,z,cS) plot(xy,cS) plot(cS,'faceColor','red','faceAlpha',0.5,'edgeColor','k') % colorize by property plot(cS,'faceColor',cS.faceAraea) {% endhighlight %}

Input

cS crystalShape
x,y coodinates
xy nx2 or nx3 coordinate matrix
PatchProperty see documentation of patch objects for manipulating the apperance, e.g. 'EdgeColor'
