defines an fibre symmetric ODF

Description

fibreODF defines a fibre symmetric ODF with respect to a crystal direction h and a specimen directions r. The shape of the ODF is defined by a kernel function.

Syntax

{% highlight matlab %} h = Miller(h,k,l,CS) r = vector3d(x,y,z); odf = fibreODF(h,r) % default halfwith 10*degree odf = fibreODF(h,r,'halfwidth',15*degree) % specify halfwidth odf = fibreODF(h,r,kernel) % specify shape odf = fibreODF(h,r,SS) % specify crystal and specimen symmetry {% endhighlight %}

Input

h Miller / vector3d crystal direction
r vector3d specimen direction
CS, SS crystal, specimen symmetry
hw halfwidth of the kernel (default -- 10Â°)
kernel kernel function (default -- de la Vallee Poussin)

Output

odf ODF
