--- title: legendre0 sidebar: function_reference_sidebar permalink: legendre0.html folder: function_reference toc: false --- legendre0 edit page

Evaluate all Legendre polynomials up to degree N in x and returns a matrix of the function values 1st dimension -> degree 2nd dimension -> x Use the recurence formula (n+1)*P_{n+1} = (2n+1)*x*P_n - n*P_{n-1}

Syntax

{% highlight matlab %} l = legendre0(N,x) {% endhighlight %}

Input

N degree
x input nodes

Output

l function evaluations
