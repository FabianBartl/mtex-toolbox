(method of [[complianceTensor_index.html,complianceTensor]])

shear modulus for an compliance tensor

Description

{% include inline_image.html file="complianceTensor.shearModulus_eq16713770968848485732.png" %}

Syntax

[GV,GR,GVRH] = shearModulus(S) % the isotropic case

E = shearModulus(S,h,u) % the anisotropic case with plane h and shear direction  u
E = shearModulus(S,[],u)
E = shearModulus(S,h,[])

Input

C elastic [[stiffnessTensor_index.html,stiffnessTensor]]
h shear plane [[vector3d_index.html,vector3d]]
u shear direction [[vector3d_index.html,vector3d]]

Output

E shear modulus
GVoigt Voigt effective shear modulus, upper bound
GReuss Reuss effective shear modulus, lower bound
GHill Hill effective shear modulus
