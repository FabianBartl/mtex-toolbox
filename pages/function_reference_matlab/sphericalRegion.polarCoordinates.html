compute polar coordinates of with respect to a spherical region

Syntax

[r,rho] = polarCoordinates(sR,v,center,ref,'param1',val1,'param2',val2)

Input

sR

sphericalRegion

v

vector3d

center

vector3d

Output

r

relative distance to the center
rho

angle with respect to the center
