heuristic to find modal orientations

Syntax

{% highlight matlab %} [modes, values] = calcModes(odf,n) {% endhighlight %}

Input

odf ODF
n number of modes

Output

modes modal orientation
values values of the ODF at the modal orientation

Options

resolution search--grid resolution
accuracy in radians

Example

{% highlight matlab %} %find the local maxima of the <SantaFe.html SantaFe> ODF mode = calcModes(SantaFe) plotPDF(SantaFe,Miller(0,0,1,mode.CS)) annotate(mode) {% endhighlight %} {% highlight plaintext %} mode = orientation size: 1 x 1 crystal symmetry : m-3m specimen symmetry: 222 Bunge Euler angles in degree phi1 Phi phi2 Inv. 296.565 48.1897 26.5651 0 {% endhighlight %} {% include inline_image.html file="ODF.calcModes_01.png" %}
