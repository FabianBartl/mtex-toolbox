define a unimodal ODF

Description

unimodalODF defines a radially symmetric, unimodal ODF with respect to a crystal orientation mod. The shape of the ODF is defined by a kernel function.

Syntax

{% highlight matlab %} mod = orientation.byEuler(phi1,Phi,phi2,CS,SS) odf = unimodalODF(mod) % default halfwidth 10 degree odf = unimodalODF(mod,'halfwidth',15*degree) % specify halfwidth odf = unimodalODF(mod,CS,SS) % specify crystal and specimen symmetry odf = unimodalODF(mod,kernel) % specify shape odf = unimodalODF(mod,'weights',weights) % specify weights for each component {% endhighlight %}

mod quaternion modal orientation
CS, SS crystal, specimen symmetry

kernel kernel function (default -- de la Vallee Poussin)
