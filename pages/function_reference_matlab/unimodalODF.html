define a unimodal ODF

Description

unimodalODF defines a radially symmetric, unimodal ODF with respect to a crystal orientation mod. The shape of the ODF is defined by a [[kernel_index.html,kernel]] function.

Syntax

mod = orientation.byEuler(phi1,Phi,phi2,CS,SS)
odf = unimodalODF(mod) % default halfwidth 10 degree
odf = unimodalODF(mod,'halfwidth',15*degree) % specify halfwidth
odf = unimodalODF(mod,CS,SS)  % specify crystal and specimen symmetry
odf = unimodalODF(mod,kernel) % specify [[kernel_index.html,kernel]] shape
odf = unimodalODF(mod,'weights',weights) % specify weights for each component

Input

mod [[quaternion_index.html,quaternion]] modal orientation
CS, SS crystal, specimen [[symmetry_index.html,symmetry]]
hw halfwidth of the kernel (default -- 10ï¿½)
kernel [[kernel_index.html,kernel]] function (default -- de la Vallee Poussin)

Output

odf [[ODF_index.html,ODF]]
