(method of vector3d)

computes the mean vector

Syntax

% average direction with respect to the first nonsingleton dimension
m = mean(v)

% average direction along dimension d
m = mean(v,d)

% average axis
m = mean(v,'antipodal')

Input

v <a href="vector3d_index.html">vector3d</a>

Options

antipodal include <a href="AxialDirectional.html">antipodal symmetry</a>

Output

m <a href="vector3d_index.html">vector3d</a>
