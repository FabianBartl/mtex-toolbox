(method of vector3d)

computes the mean vector

Syntax

% average direction with respect to the first nonsingleton dimension
m = mean(v)

% average direction along dimension d
m = mean(v,d)

% average axis
m = mean(v,'antipodal')

Input

v

vector3d

param,val Parameters and values that control mean
Parameter Description
'antipodal'

Output

m
