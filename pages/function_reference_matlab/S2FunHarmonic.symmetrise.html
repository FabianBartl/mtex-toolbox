(method of S2FunHarmonic)

symmetrises a function with respect to a symmetry or a direction

Syntax

% symmetrise with respect to a crystal or specimen symmetry
sFs = symmetrise(sF,cs)
sFs = symmetrise(sF,ss)
[sFs,psi] = symmetrise(sF,d)

% symmetrise with respect to an axis
sFs = symmetrise(sF,d)

Input

sF <a href="S2FunHarmonic_index.html">S2FunHarmonic</a>
cs,ss <a href="crystalSymmetry_index.html">crystalSymmetry</a>, <a href="specimenSymmetry_index.html">specimenSymmetry</a>
d <a href="vector3d_index.html">vector3d</a>

Output

sFs symmetrised <a href="S2FunHarmonic_index.html">S2FunHarmonic</a>
psi <a href="S2Kernel_index.html">S2Kernel</a>
