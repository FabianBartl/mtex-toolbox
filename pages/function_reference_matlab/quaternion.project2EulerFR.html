projects quaternions to a fundamental region

Syntax

{% highlight matlab %} [phi1,Phi,phi2] = project2EulerFR(q,CS) % Bunge fundamental region [alpha,beta,gamma] = project2EulerFR(q,CS,'Matthies') % Matthies fundamental region [phi1,Phi,phi2] = project2EulerFR(q,CS1,CS2) % misorientations {% endhighlight %}

Input

q quaternion
CS1, CS2 crystal symmetry

Output

phi1,Phi,phi2 Euler angles
