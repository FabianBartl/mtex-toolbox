convert x,y coordinates into indeces of ebsd

Syntax

{% highlight matlab %} ind = xy2ind(ebsd,x,y) ebsd(ind) {% endhighlight %} {% highlight matlab %} [i,j] = ind = xy2ind(ebsd,x,y) ebsd(i,j) {% endhighlight %}

Input

ebsd @EBSDSquare
x,y spatial coordinates

Output

ind index to @EBSDSquare
i,j indeces to @EBSDSquare
