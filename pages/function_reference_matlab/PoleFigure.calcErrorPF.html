error polefigure between meassured and recalculated pole figures

returns a PoleFigure with valuess given as the difference between the meassured intensities and the recalculated ODF or between two meassured PoleFigure.

Syntax

{% highlight matlab %} pf = calcErrorPF(pfmeas,pfcalc) % pf = calcErrorPF(pfmeas,odf) % {% endhighlight %}

Input

pfmeas meassured PoleFigure
pfcalc recalculated PoleFigure
odf recalculated ODF

Output

pf PoleFigure

Options

RP RP value (default)
l1 l1 error
l2 l2 error
