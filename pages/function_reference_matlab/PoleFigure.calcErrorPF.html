(method of PoleFigure)

error polefigure between meassured and recalculated pole figures

returns a PoleFigure with valuess given as the difference between the meassured intensities and the recalculated ODF or between two meassured PoleFigure.

Syntax

pf = calcErrorPF(pfmeas,pfcalc) %
pf = calcErrorPF(pfmeas,odf)    %

Input

pfmeas

meassured PoleFigure

pfcalc

recalculated PoleFigure

odf

recalculated ODF

param,val Parameters and values that control calcErrorPF
Parameter Description
'RP'

RP value (default)
'l1'

l1 error
'l2'

l2 error

Output

pf

PoleFigure
