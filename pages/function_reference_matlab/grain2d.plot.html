colorize grains

Syntax

{% highlight matlab %} plot(grains) % colorize by phase plot(grains,property) % colorize by property plot(grains,cS) % visualize crystal shape {% endhighlight %}

Input

grains grain2d
cS crystalShape
PatchProperty see documentation of patch objects for manipulating the apperance, e.g. 'EdgeColor'

Options

noBoundary do not plot boundaries
displayName name used in legend
