find rotation rot such that l = rot * r

Description

Find the rotation that best maps all the vectors r onto the vectors l.

Syntax

{% highlight matlab %} rot = rotation.fit(r,l) rot = rotation.fit(r,l,'weights',w) {% endhighlight %}

Input

r, l vector3d

Output

rot rotation
