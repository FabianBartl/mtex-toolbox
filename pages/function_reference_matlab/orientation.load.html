(method of orientation)

import orientation data from data files

Description

orientation.load is a high level method for importing orientations from column aligned text files or excel spread sheets. In those cases it is neccesary to tell MTEX the column positions of the spatial coordinates, the phase information as well as Euler angles.

Syntax

ori = orientation.load(fname,cs,'ColumnNames',{'phi1','Phi','phi2'})

Input

fname filename
cs <a href="crystalSymmetry_index.html">crystalSymmetry</a>
ss <a href="specimenSymmetry_index.html">specimenSymmetry</a> (optional)

Options

columnNames names of the colums to be imported, mandatory are euler 1, euler 2, euler 3
columns postions of the columns to be imported
radians treat input in radiand
delimiter delimiter between numbers
header number of header lines
Bunge [phi1 Phi phi2] Euler angle in Bunge convention (default)
passive

Output

ori <a href="orientation_index.html">orientation</a>
