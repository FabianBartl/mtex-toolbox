rotational axis, roational angle to Quaternion

Decription defines a rotation by a rotational axis and a roational angle

Syntax

{% highlight matlab %} q = achs2quat(v,omega) {% endhighlight %}

Input

v rotational axis (vector3d)
omega rotational angle

Output

q quaternion
