(method of orientation)

define orientations by pairs of vectors

Description

Define an orientation that maps u1 onto v1 and u2 onto v2

Syntax

ori = orientation.map(u1,v1)
ori = orientation.map(u1,v1,u2,v2)
ori = orientation.map(u1,v1,u2,v2,CS,SS)

Input

u1, v1, u2, v2 <a href="vector3d_index.html">vector3d</a> <a href="Miller_index.html">Miller</a>
CS <a href="crystalSymmetry_index.html">crystalSymmetry</a>
SS <a href="specimenSymmetry_index.html">specimenSymmetry</a>

Output

ori <a href="orientation_index.html">orientation</a>
