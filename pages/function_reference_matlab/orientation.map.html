define orientations by pairs of vectors

Description

Define an orientation that maps u1 onto v1 and u2 onto v2

Syntax

{% highlight matlab %} ori = orientation.map(u1,v1) ori = orientation.map(u1,v1,u2,v2) ori = orientation.map(u1,v1,u2,v2,CS,SS) {% endhighlight %}

Input

u1, v1, u2, v2 vector3d Miller
CS crystalSymmetry
SS specimenSymmetry

Output

ori orientation
