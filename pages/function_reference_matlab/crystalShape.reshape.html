Syntax

   B = reshape(A,sz)
   B = reshape(A,sz1,...,szN)

Input

A Input array
vector | matrix | multidimensional array
sz Output size
row vector of integers
sz1,...,szN Size of each dimension
two or more integers | [] (optional)

Output

B Reshaped array
vector | matrix | multidimensional array | cell array
