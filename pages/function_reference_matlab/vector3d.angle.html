angle between two vectors

Syntax

{% highlight matlab %} omega = angle(v1,v2) omega = angle(v1,v2,'antipodal') omega = angle(v1,v2,N) {% endhighlight %}

Input

v1, v2 vector3d
N vector3d normal to plane

Output

omega double

Options

antipodal include antipodal symmetry
