(method of quaternion)

the logarithmic map that translates a rotation into a rotation vector

Syntax

v = log(q) % rotation vector with reference to the identical rotation
v = log(q,q_ref) % rotation vector with reference q_ref

v = loq(q,'right')
v = loq(q,'left')

Input

q <a href="quaternion_index.html">quaternion</a>
q_ref <a href="quaternion_index.html">quaternion</a>

Output

v <a href="vector3d_index.html">vector3d</a>
