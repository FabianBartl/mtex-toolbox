import Tensor data

Description

loadTensor_mod is a high level method for importing Tensor data from external files. Material Properties Open Database

Syntax

{% highlight matlab %} T = loadTensor_mpod(fname,cs,ss) {% endhighlight %}

Input

fname filename
cs, ss crystal, specimen symmetry (optional)

Example

download form MPOD

{% highlight matlab %} T = loadTensor_mpod(1000055) T = loadTensor_mpod('1000055.mpod') {% endhighlight %} {% highlight plaintext %} Deviation:36 Deviation:0.3 T = 2×1 cell array {1×1 tensor} {1×1 tensor} Deviation:36 Deviation:0.3 T = 2×1 cell array {1×1 tensor} {1×1 tensor} {% endhighlight %}
