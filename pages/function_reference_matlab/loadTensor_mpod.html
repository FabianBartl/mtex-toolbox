Description

loadTensor_mod is a high level method for importing Tensor data from external files. Material Properties Open Database

Syntax

pf = loadTensor_mpod(fname,cs,ss,<options>)

Input

fname

filename
cs, ss

crystal, specimen [[symmetry_index.html,symmetry]] (optional)

Example

download form MPOD

T = loadTensor_mpod(1000055)
T = loadTensor_mpod('1000055.mpod')
