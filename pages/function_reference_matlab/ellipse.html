annotate an ellipse in a spherical plot

Syntax

h = ellipse(r,a,b,varargin)

Input

r <a href="rotation_index.html">rotation</a> i.e. matrix of three orthogonal vectors specifying ellipse position
a,b long/short half axes of ellipse in radian
