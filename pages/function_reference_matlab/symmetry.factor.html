factorizes s1 and s2 into l, d, r such that s1 = l * d and s2 = d * r

Syntax

{% highlight matlab %} [l,d,r] = factor(s1,s2) {% endhighlight %}

Input

s1, s2 symmetry
l, r, d quaternion
