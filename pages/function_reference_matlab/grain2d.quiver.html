plot directions at grain centers

Syntax

{% highlight matlab %} quiver(grains,dir,'linecolor','r') {% endhighlight %} {% highlight matlab %} mtexdata fo grains = calcGrains(ebsd('indexed')) quiver(grains,grains.meanRotation.axis,'color','r') {% endhighlight %}

Input

grains grain2d
dir vector3d

Options

antipodal plot directions or axes
maxHeadSize size of the arrow
