ratio of orientations with a certain orientation

Description

returns the ratio of mass of the odf that is within a certain distance from a given fibre

Syntax

{% highlight matlab %} v = fibreVolume(odf,h,r,radius) {% endhighlight %}

Input

odf ODF
h Miller
r vector3d
radius double

Options

resolution resolution of discretization
