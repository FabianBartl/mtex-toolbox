select EBSD data by spatial coordinates

Syntax

{% highlight matlab %} map = findByLocation( ebsd, xy, y ) {% endhighlight %}

Input

ebsd EBSD
xy list of [x(:) y(:)] coordinates, respectively [x(:) y(:) z(:)]

Output

ebsd EBSD subset

Example

{% highlight matlab %} mtexdata forsterite plotx2east plot(ebsd) p = [10000 5000] %ginput(1) g = findByLocation(ebsd,p) {% endhighlight %} {% highlight plaintext %} saving data to /home/hielscher/mtex/master/data/forsterite.mat ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 58485 (24%) notIndexed 1 152345 (62%) Forsterite LightSkyBlue mmm 2 26058 (11%) Enstatite DarkSeaGreen mmm 3 9064 (3.7%) Diopside Goldenrod 12/m1 X||a*, Y||b*, Z||c Properties: bands, bc, bs, error, mad Scan unit : um X x Y x Z : [0 36550] x [0 16750] x [0 0] Normal vector: (0,0,1) p = 10000 5000 g = 73401 {% endhighlight %}
