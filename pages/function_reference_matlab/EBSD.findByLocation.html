select EBSD data by spatial coordinates

Syntax

{% highlight matlab %} map = findByLocation( ebsd, xy ) {% endhighlight %}

Input

ebsd EBSD
xy list of [x(:) y(:)] coordinates, respectively [x(:) y(:) z(:)]

Output

ebsd EBSD subset

Example

{% highlight matlab %} mtexdata fo plotx2east plot(ebsd) p = [10000 5000] %ginput(1) g = findByLocation(ebsd,p) {% endhighlight %} {% highlight plaintext %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 58485 (24%) notIndexed 1 152345 (62%) Forsterite LightSkyBlue mmm 2 26058 (11%) Enstatite DarkSeaGreen mmm 3 9064 (3.7%) Diopside Goldenrod 12/m1 X||a*, Y||b*, Z||c Properties: bands, bc, bs, error, mad, x, y Scan unit : um p = 10000 5000 g = 73401 {% endhighlight %} {% include inline_image.html file="EBSD.findByLocation_01.png" %}
