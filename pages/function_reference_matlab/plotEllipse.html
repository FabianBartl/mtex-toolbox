plot multiple ellipses

Syntax

{% highlight matlab %} [omega,a,b] = principalComponents(grains); plotEllipse(grains.centroid,a,b,omega,'lineColor','r') {% endhighlight %}

Input

cxy center of the ellipse
a,b length of the half axes
omega orientation of the ellipse

Options

lineColor colorspec
