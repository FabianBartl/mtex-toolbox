estimate grain boundary plane distribution from 2d sections

Syntax

{% highlight matlab %} % compute gbnd in crystal coordinates gbnd = calcGBND(gB,ebsd) gbnd = calcGBND(gB,grains) {% endhighlight %} {% highlight matlab %} % use a specific halfwidth gbnd = calcGBND(gB,ebsd,'halfwidth',10*degree) {% endhighlight %}

Input

gB grainBoundary
ebsd EBSD
grains grain2d

Output

gpnd S2FunHarmonic

Options

halfwidth used for kernel density estimation

References

  • D.M. Saylor, G.S. Rohrer: Determining crystal habits from observations of planar sections in J. Am. Ceram. Soc., 85(11):2799–2804, 2002.
  • R. Hielscher, R. Kilian, E. Wünsche: Efficient computation of the grain boundary normal distribution from two dimensional EBSD data, not yet published.
