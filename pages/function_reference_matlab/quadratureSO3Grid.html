--- title: quadratureSO3Grid sidebar: function_reference_sidebar permalink: quadratureSO3Grid.html folder: function_reference toc: false --- quadratureSO3Grid edit page

Calculate the nodes and weights of specific quadrature grids on SO(3) 'ClenshawCurtis' - combinate the Gauss quadrature in 1st and 3rd Euler angle (alpha, gamma) with Clenshaw Curtis quadrature in 2nd Euler angle (beta)

Syntax

{% highlight matlab %} [SO3G, W, M2] = quadratureSO3rid(M, 'ClenshawCurtis') quadrature grid of type Clenshaw Curtis {% endhighlight %}
