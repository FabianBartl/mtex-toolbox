(method of rotation)

define rotations by pairs of vectors

Description

Define a rotation that maps u1 onto v1 and u2 onto v2

Syntax

{% highlight matlab %} % an arbitrary rotation that maps u1 parallel to v1 rot = rotation(u1,v1) {% endhighlight %} {% highlight matlab %} % a rotation that maps u1 parallel to v1 and u2 parallel to v2 rot = rotation(u1,v1,u2,v2) {% endhighlight %}

Input

u1, v1, u2, v2 vector3d

Output

rot rotation
