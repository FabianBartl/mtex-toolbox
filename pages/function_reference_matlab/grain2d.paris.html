Percentile Average Relative Indented Surface

the paris (Percentile Average Relative Indented Surface) is shape parameter that measures the convexity of a grain

Syntax

p = paris(grains)

Input

grains <a href="grain2d_index.html">grain2d</a>

Output

p double
