hierarchical clustering of rotations and vectors

Syntax

{% highlight matlab %} [c,center] = doHCluster(ori,'numCluster',n) [c,center] = doHCluster(ori,'maxAngle',omega) {% endhighlight %}

Input

ori orientation
n number of clusters
omega maximum angle

Output

c list of clusters
center center of the clusters

Example

{% highlight matlab %} % generate orientation clustered around 5 centers cs = crystalSymmetry('m-3m'); center = orientation.rand(5,cs); odf = unimodalODF(center,'halfwidth',5*degree) ori = odf.discreteSample(3000); {% endhighlight %} {% highlight plaintext %} odf = SO3FunRBF (m-3m → xyz) multimodal components kernel: de la Vallee Poussin, halfwidth 5° center: 5 orientations Bunge Euler angles in degree phi1 Phi phi2 weight 286.058 120.155 72.7244 0.2 340.837 47.4666 241.324 0.2 155.812 121.89 109.082 0.2 252.539 97.1975 328.226 0.2 317.871 35.623 109.985 0.2 {% endhighlight %} {% highlight matlab %} % find the clusters and its centers tic; [c,centerRec] = calcCluster(ori,'method','hierarchical','numCluster',5); toc {% endhighlight %} {% highlight plaintext %} Elapsed time is 2.942310 seconds. {% endhighlight %} {% highlight matlab %} % visualize result oR = fundamentalRegion(cs) plot(oR) {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: 432 max angle: 62.7994° face normales: 14 vertices: 24 {% endhighlight %}
