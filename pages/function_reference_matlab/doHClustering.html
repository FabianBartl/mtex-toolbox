hierarchical clustering of rotations and vectors

Syntax

{% highlight matlab %} [c,center] = doHCluster(ori,'numCluster',n) [c,center] = doHCluster(ori,'maxAngle',omega) {% endhighlight %}

Input

ori orientation
n number of clusters
omega maximum angle

Output

c list of clusters
center center of the clusters

Example

{% highlight matlab %} % generate orientation clustered around 5 centers cs = crystalSymmetry('m-3m'); center = orientation.rand(5,cs); odf = unimodalODF(center,'halfwidth',5*degree) ori = odf.discreteSample(3000); {% endhighlight %} {% highlight plaintext %} odf = SO3FunRBF (m-3m → xyz) multimodal components kernel: de la Vallee Poussin, halfwidth 5° center: 5 orientations Bunge Euler angles in degree phi1 Phi phi2 weight 95.1308 87.1324 51.6591 0.2 157.933 84.0798 23.5626 0.2 177.216 96.5345 142.363 0.2 177.603 132.704 143.501 0.2 337.534 89.1795 249.202 0.2 {% endhighlight %} {% highlight matlab %} % find the clusters and its centers tic; [c,centerRec] = calcCluster(ori,'method','hierarchical','numCluster',5); toc {% endhighlight %} {% highlight plaintext %} Elapsed time is 3.064437 seconds. {% endhighlight %} {% highlight matlab %} % visualize result oR = fundamentalRegion(cs) plot(oR) {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: 432 max angle: 62.7994° face normales: 14 vertices: 24 {% endhighlight %}
