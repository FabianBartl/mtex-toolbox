compute the Wigner d and Wigner D functions

Syntax

{% highlight matlab %} Dmn = Wigner_D(L,rot) dmn = Wigner_D(L,beta) {% endhighlight %}

Input

L harmonic degree
rot rotation
beta second Euler angle

Output

Dmn Wigner D matrix D_L^(m,n)
dmn Wigner d matrix d_L^(m,n)
