spatial EBSD plot

Syntax

{% highlight matlab %} % colorize according to phase plot(ebsd) {% endhighlight %} {% highlight matlab %} % colorize according to arbitrary value - here MAD plot(ebsd,ebsd.mad) {% endhighlight %} {% highlight matlab %} % colorize according to orientation plot(ebsd('phaseName'),ebsd('phaseName').orientation) {% endhighlight %} {% highlight matlab %} % colorize according to custom color oM = ipfColorKey(ebsd('phaseName')) color = oM.orientation2color(ebsd('phaseName').orientations); plot(ebsd('phaseName'),color) {% endhighlight %} {% highlight matlab %} % specify the color directly and show in Legend badMAD = ebsd.mad > 1; plot(ebsd(badMAD),'faceColor','black,'DisplayName','bad values') {% endhighlight %}

Input

ebsd EBSD
color length(ebsd) x 3 vector of RGB values

Options

micronbar 'on'/'off'
DisplayName add a legend entry

Flags

points plot dots instead of unitcells
exact plot exact unitcells, even for large maps

Example

{% highlight matlab %} mtexdata forsterite plot(ebsd) {% endhighlight %} {% highlight plaintext %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 58485 (24%) notIndexed 1 152345 (62%) Forsterite LightSkyBlue mmm 2 26058 (11%) Enstatite DarkSeaGreen mmm 3 9064 (3.7%) Diopside Goldenrod 12/m1 X||a*, Y||b*, Z||c Properties: bands, bc, bs, error, mad, x, y Scan unit : um {% endhighlight %}
