select EBSD data along line segments

Syntax

{% highlight matlab %} ebsdLine = spatialProfile(ebsd,[xStart,xEnd],[yStart yEnd]) {% endhighlight %} {% highlight matlab %} [ebsdLine,dist] = spatialProfile(ebsd,x,y) {% endhighlight %} {% highlight matlab %} xy = ginput(2) [ebsdLine,dist] = spatialProfile(ebsd,xy) {% endhighlight %}

Input

ebsd EBSD
xStart, xEnd, yStart, yEnd double
x, y coordinates of the line segments
xy list of spatial coordinates |[x(:) y(:)]|

Output

ebsdLine EBSD restrcited to the line of interest
dist double distance along the line to the initial point

Example

{% highlight matlab %} % import data mtexdata twins {% endhighlight %} {% highlight plaintext %} ebsd = EBSD Phase Orientations Mineral Color Symmetry Crystal reference frame 0 46 (0.2%) notIndexed 1 22833 (100%) Magnesium LightSkyBlue 6/mmm X||a*, Y||b, Z||c* Properties: bands, bc, bs, error, mad Scan unit : um X x Y x Z : [0 50] x [0 41] x [0 0] Normal vector: (0,0,1) {% endhighlight %} {% highlight matlab %} % plot data plot(ebsd('indexed'),ebsd('indexed').orientations) {% endhighlight %}
