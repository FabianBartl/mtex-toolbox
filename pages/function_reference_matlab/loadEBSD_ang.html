read EDAX *.ang file

Syntax

{% highlight matlab %} % change x and y values such that spatial and Euler reference frame % coincide, i.e., rotate them by 180 degree ebsd = loadEBSD_ang(fname,'convertSpatial2EulerReferenceFrame') {% endhighlight %} {% highlight matlab %} % change the Euler angles such that spatial and Euler reference frame % coincide, i.e., rotate them by 180 degree ebsd = loadEBSD_ang(fname,'convertEuler2SpatialReferenceFrame') {% endhighlight %}

Input

fname file name

Flags

convertSpatial2EulerReferenceFrame
convertEuler2SpatialReferenceFrame
