--- title: plotSO3Grid last_updated: 26-Mar-2019 sidebar: function_reference_sidebar permalink: plotSO3Grid.html folder: function_reference toc: false --- plotSO3Gridedit page

give a regular grid in orientation space

Syntax

% sections according phi2 angle
[S3G,S2G,phi2] = plotSO3Grid(CS,SS,'phi2')

% sections according rotational axis / angle
[S3G,axes,omega,scaling] = plotSO3Grid(CS1,CS2,'axisAngle')

Input

CS

crystalSymmetry

SS

specimenSymmetry

Output

S3G

SO3Grid

S2G,axes

list of vector3d, the plotting directions
phi2

double
scaling

double
