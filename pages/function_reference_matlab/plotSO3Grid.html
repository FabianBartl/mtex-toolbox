--- title: plotSO3Grid last_updated: 07-Oct-2019 sidebar: function_reference_sidebar permalink: plotSO3Grid.html folder: function_reference toc: false --- plotSO3Gridedit page

give a regular grid in orientation space

Syntax

% sections according phi2 angle
[S3G,S2G,phi2] = plotSO3Grid(CS,SS,'phi2')

% sections according rotational axis / angle
[S3G,axes,omega,scaling] = plotSO3Grid(CS1,CS2,'axisAngle')

Input

CS <a href="crystalSymmetry_index.html">crystalSymmetry</a>
SS <a href="specimenSymmetry_index.html">specimenSymmetry</a>

Output

S3G <a href="SO3Grid_index.html">SO3Grid</a>
S2G,axes list of <a href="vector3d_index.html">vector3d</a>, the plotting directions
phi2 double
scaling double
