(method of [[ODF_index.html,ODF]])

plot odf along a fibre

Syntax

plotFibre(odf,f);

Input

odf [[ODF_index.html,ODF]]
f [[fibre_index.html,fibre]]

Options

resolution resolution of each plot

Example

odf = SantaFe;
f = fibre.gamma(odf.CS,odf.SS)
plotFibre(SantaFe,f)

 
f = fibre  
 size: 1 x 1
 crystal symmetry:  m-3m
 specimen symmetry: 222
 o1: (180°,55°,45°)
 o2: (270°,55°,45°)
