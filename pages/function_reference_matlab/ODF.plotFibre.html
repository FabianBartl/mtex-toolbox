plot odf along a fibre

Syntax

{% highlight matlab %} plotFibre(odf,f); {% endhighlight %}

Input

odf ODF
f fibre

Options

resolution resolution of each plot

Example

{% highlight matlab %} odf = SantaFe; f = fibre.gamma(odf.CS,odf.SS) plotFibre(SantaFe,f) {% endhighlight %} {% highlight plaintext %} f = fibre (m-3m → xyz (222)) h || r: (111) || (0,0,1) o1 -> o2: (180°,54.7°,45°) -> (270°,54.7°,45°) {% endhighlight %}
