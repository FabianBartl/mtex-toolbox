(method of ODF)

plot odf along a fibre

Syntax

{% highlight matlab %} plotFibre(odf,f); {% endhighlight %}

Input

odf ODF
f fibre

Options

resolution resolution of each plot

Example

{% highlight matlab %} odf = SantaFe; f = fibre.gamma(odf.CS,odf.SS) plotFibre(SantaFe,f) {% endhighlight %} {% highlight plaintext %} f = fibre size: 1 x 1 crystal symmetry: m-3m specimen symmetry: 222 o1: (180°,54.7°,45°) o2: (270°,54.7°,45°) {% endhighlight %} {% include inline_image.html file="ODF.plotFibre_01.png" %}
