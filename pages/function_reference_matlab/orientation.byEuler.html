define orientations by Euler angles

Syntax

{% highlight matlab %} ori = orientation.byEuler(phi1,Phi,phi2,CS,SS) % Bunge convention ori = orientation.byEuler(alpha,beta,gamma,'ZYZ',CS,SS) % Matthies convention ori = orientation.byEuler(phi1,Phi,phi2,'Kocks',CS,SS) % Kocks convention {% endhighlight %}

Input

phi1, Phi, phi2 Euler angles in radiant
CS crystalSymmetry
SS specimenSymmetry

Output

ori orientation

Flags

Bunge, ZXZ
ABG, Matthies, ZYZ
Roe
Kocks
Canova
