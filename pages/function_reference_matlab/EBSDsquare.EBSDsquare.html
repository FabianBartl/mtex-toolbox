(method of EBSDsquare)

EBSD data on a rectangular grid. In contrast to arbitrary EBSD data the values are stored in a matrix.

Syntax

ebsd = EBSDsquare(rot,phaseId,phaseMap,CSList,dxy,'param1',val1,'param2',val2)
