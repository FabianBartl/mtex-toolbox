grain area weighted histogram

Syntax

{% highlight matlab %} histogram(grains) histogram(grains,n) % specify the number of bins {% endhighlight %} {% highlight matlab %} % use an arbitrary property for the histogramm histogram(grains,grains.equivalentRadius) {% endhighlight %} {% highlight matlab %} h = histogram(grains) {% endhighlight %}

Input

grains grain2d
n number of bins, default ist 15

Output

h handle to the histogram graphics object
