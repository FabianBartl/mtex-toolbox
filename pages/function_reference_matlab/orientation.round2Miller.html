find lattice alignements for arbitrary orientations and misorientations

Description

Given an orienation ori find [hkl](uvw) such that ori * [hkl] = Z and ori * (uvw) = X.

Given a misorientation mori find corresponding face normals n1, n2 and crystal directions d1, d2, i.e., such that mori * n1 = n2 and mori * d1 = d2.

Syntax

{% highlight matlab %} [uvw,hkl] = round2Miller(ori) {% endhighlight %} {% highlight matlab %} [n1,n2,d1,d2] = round2Miller(mori) [n1,n2,d1,d2] = round2Miller(mori,'penalty',0.01) [n1,n2,d1,d2] = round2Miller(mori,'maxIndex',6) {% endhighlight %}

Input

ori orientation
mori misorientation

Output

uvw,hkl Miller
n1,n2,d1,d2 Miller

Example

{% highlight matlab %} % revert sigma3 misorientation relationship [n1,n2,d1,d2] = round2Miller(CSL(3,crystalSymmetry('432'))) {% endhighlight %} {% highlight plaintext %} n1 = Miller symmetry: 432 h k l 1 1 1 n2 = Miller symmetry: 432 h k l 1 1 1 d1 = Miller symmetry: 432 u v w 0 1 -1 d2 = Miller symmetry: 432 u v w -1 1 0 {% endhighlight %} {% highlight matlab %} % revert back Bain misorientation ship cs_alpha = crystalSymmetry('m-3m', [2.866 2.866 2.866], 'mineral', 'Ferrite'); cs_gamma = crystalSymmetry('m-3m', [3.66 3.66 3.66], 'mineral', 'Austenite'); mori = orientation.Bain(cs_alpha,cs_gamma) [n_gamma,n_alpha,d_gamma,d_alpha] = round2Miller(mori) {% endhighlight %} {% highlight plaintext %} mori = misorientation (Ferrite -> Austenite) Bunge Euler angles in degree phi1 Phi phi2 Inv. 0 45 0 0 n_gamma = Miller mineral: Ferrite (m-3m) h k l 1 0 0 n_alpha = Miller mineral: Austenite (m-3m) h k l 1 0 0 d_gamma = Miller mineral: Ferrite (m-3m) u v w 0 1 0 d_alpha = Miller mineral: Austenite (m-3m) u v w 0 1 1 {% endhighlight %}
