(method of slipSystem)

trace of the slip plane in the surface normal to n

Syntax

{% highlight matlab %} t = trace(sS) % trace in the xy surface t = trace(sS,n) % trace in the surface normal to n {% endhighlight %}

Input

sS list of slipSystem
n vector3d normal vector

Output

t vector3d
