(method of slipSystem)

trace of the slip plane in the surface normal to n

Syntax

t = trace(sS) % trace in the xy surface
t = trace(sS,n) % trace in the surface normal to n

Input

sS list of <a href="slipSystem_index.html">slipSystem</a>
n <a href="vector3d_index.html">vector3d</a> normal vector

Output

t <a href="vector3d_index.html">vector3d</a>
